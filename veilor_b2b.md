# VEILOR B2B 서비스 검토 문서

> 작성일: 2026-04-13  
> 목적: 스포츠/엔터테인먼트 기업 대상 B2B 멘탈관리 서비스 진출 가능성 검토

---

## 1. 배경

베일러(VEILOR)는 현재 B2C 멘탈관리 플랫폼으로 운영 중.  
스포츠엔터테인먼트 + 아이돌엔터테인먼트 회사를 운영하는 잠재 고객으로부터 소속 선수/아이돌의 멘탈관리 서비스 제공 가능 여부 문의를 받음.

---

## 1-1. 핵심 가치 제안: 오프라인 코치 비용을 베일러로 대체한다

> **베일러 B2B의 구매 동기는 단순하다.**  
> 지금 쓰고 있는 오프라인 전담 코치/상담사 비용을 줄이거나 없앤다.  
> 고정 인건비 구조를 플랫폼 구독료로 전환한다.

---

### 1-1-1. 현재 고객사의 오프라인 멘탈관리 비용 구조

**스포츠팀 (프로구단 기준)**

| 항목 | 비용 | 비고 |
|---|---|---|
| 전담 스포츠 심리 상담사 고용 | 400~700만원/월 | 전임 기준, 정규직 or 촉탁 |
| 외부 멘탈 코치 계약 | 200~400만원/월 | 세션 단위 계약, 월 8~12회 |
| 해외 원정·전지훈련 동행 비용 | 별도 | 항공·숙박 포함 시 연 500~1,000만원 |
| **실질 연간 비용** | **7,200만~1억 3,200만원** | 1인 전담 기준 |
| 실제 이용률 | 선수단 15~30% | 낙인 효과로 자발적 이용 저조 |

**K팝 소속사 (중형 기준: 아티스트 10~30명)**

| 항목 | 비용 | 비고 |
|---|---|---|
| 외부 심리상담사 계약 | 150~300만원/월 | 주 1~2회 방문 계약 |
| 1인당 세션 비용 | 10~15만원/회 | 개인 세션 기준 |
| 실제 이용 멤버 수 | 전체의 20~35% | 데뷔 멤버 중심, 트레이니 미포함 |
| **실질 연간 비용** | **1,800만~3,600만원** | 중형 소속사 기준 |
| 트레이니 커버 여부 | 대부분 미포함 | 예산·계약 범위 외 |

**일반 기업 (300인 이상)**

| 항목 | 비용 | 비고 |
|---|---|---|
| EAP 외부 위탁 계약 | 인당 5~12만원/년 | 전통 EAP, 실이용률 7% |
| 사내 상담사 고용 | 300~500만원/월 | 300인 이상 의무화 논의 중 |
| 1회 외부 상담 지원 | 5~8만원/회 지원 | 복지포인트 형태 |
| **실질 연간 비용 (300인)** | **1,500만~3,600만원** | EAP + 상담사 고용 합산 |
| 실제 이용률 | 7~11% | 낙인 효과 |

### 1-1-2. 베일러 도입 후 비용 구조 전환

**비교 시뮬레이션 — 스포츠팀 30명**

```
현재 구조:
  전담 코치 1인 고용     600만원/월
  ───────────────────────────────
  연간 총비용            7,200만원
  실제 커버 인원         4~9명 (이용률 15~30%)
  1인당 실질 비용        800만~1,800만원/년

베일러 도입 후:
  베일러 B2B 그로스 플랜  69,000원 × 30명 = 207만원/월
  + 코치 온디맨드 세션    필요 시만 과금 (세션당 별도)
  ───────────────────────────────
  연간 플랫폼 비용        2,484만원
  실제 커버 인원         30명 전원 (일상 체크인)
  1인당 비용             82만 8천원/년

절감액:    4,716만원/년 (65% 절감)
커버리지:  4~9명 → 30명 전원
```

**비교 시뮬레이션 — K팝 소속사 20명**

```
현재 구조:
  외부 상담사 월 계약     200만원/월
  ───────────────────────────────
  연간 총비용            2,400만원
  실제 커버 인원         4~7명 (이용률 20~35%)
  트레이니 커버          없음

베일러 도입 후:
  아티스트 그로스 플랜    69,000원 × 20명 = 138만원/월
  트레이니 베이직 추가    59,000원 × 10명 = 59만원/월
  ───────────────────────────────
  연간 플랫폼 비용        2,364만원
  실제 커버 인원         아티스트 20명 + 트레이니 10명 전원

비용 거의 동일, 커버 인원 3~4배 확장
트레이니 조기 관리 시작 → 데뷔 후 케어 연속성 확보
```

**비교 시뮬레이션 — 기업 100명**

```
현재 구조:
  EAP 외부 위탁          7만원 × 100명 = 700만원/년
  사내 상담사 1인         400만원/월 = 4,800만원/년
  ───────────────────────────────
  연간 총비용            5,500만원
  실제 이용률            7% (7명)

베일러 도입 후:
  베일러 기업 플랜        69,000원 × 100명 = 690만원/월
  ───────────────────────────────
  연간 플랫폼 비용        8,280만원
  실제 참여율 목표        70%+ (30초 체크인 UX)

  단, 사내 상담사 고용 불필요 → 4,800만원 절감
  순비용: 8,280 - 4,800 = 3,480만원/년
  → 현재(5,500만원) 대비 36% 절감 + 이용률 10배
```

### 1-1-3. 베일러가 오프라인을 완전 대체하지 않는 경우

> 베일러는 **일상 퍼포먼스 관리 레이어**를 담당한다.  
> 오프라인 전문가는 **위기 개입·심층 케어 레이어**로 역할이 재정의된다.

```
기존: 오프라인 코치 1인이 전부 담당
  → 일상 체크인 + 위기 대응 + 심층 상담 + 그룹 세션 모두

베일러 도입 후: 역할 분리
  베일러          일상 체크인, 루틴 관리, 코치 온디맨드 연결
  오프라인 코치   위기 개입, 심층 케어, 베일러 에스컬레이션 케이스만 담당

결과:
  오프라인 코치 1인 full-time → part-time (주 2회) 전환 가능
  비용: 600만원/월 → 150~200만원/월
  베일러 + 파트타임 코치 총비용: 207만 + 175만 = 382만원/월
  → 기존 600만원 대비 36% 절감, 커버리지는 전원으로 확대
```

## 2. 경쟁사 현황

### 2-1. 주요 플레이어

| 회사 | 서비스 | B2B 현황 | 특이사항 |
|---|---|---|---|
| **마인드카페** (아토머스) | 심리상담, AI 멘탈케어 | ✅ 적극 확장 중 | EAP 전문기업 '밝음' 인수. 조직 심리진단 리포트 제공. 연매출 200억 목표. AI 기반 SaaS 전환 추진 |
| **트로스트** (넛지헬스케어) | 비대면 텍스트 상담, AI 셀프케어 | ✅ EAP 전문 사이트 별도 운영 (`trosteap.co.kr`) | 2025년 4월 캐시워크(넛지헬스케어)에 인수. EAP 올인원 솔루션, 무제한 임직원 심리상담 제공 |
| **힐다임** | 기업상담(EAP) | ✅ B2B 전문 | 중소기업 대상 EAP |
| **마음이음** | 온라인 상담 | 정보 미확인 | 스포츠/엔터 특화 B2B는 미확인 |

### 2-2. 시장 흐름

- **EAP(직원지원프로그램) 도입이 빠르게 확산 중**: 중대재해처벌법 시행 이후 기업의 근로자 정신건강 관리 의무 인식 강화
- **대형 플레이어들이 B2B로 피벗 중**: 마인드카페(B2C→B2B+B2G 확장), 트로스트(대기업에 인수되어 EAP 통합)
- **엔터/스포츠 특화 B2B는 현재 공백**: 현재까지 아이돌사 또는 스포츠구단에 특화된 멘탈관리 SaaS 플레이어는 확인되지 않음 — **이 지점이 베일러의 진입 기회**

### 2-3. 공공 지원 사례 (시장 수요 근거)

- 문화체육관광부 산하 **예술인 개인 심리상담 지원 사업** 운영 중 (2025년 2~10월)  
  → 예술활동증명 예술인 대상, 국가가 직접 멘탈 지원 필요성을 인정한 근거
- 출연연(정부출연연구기관) **힐링톡 심리상담 프로그램** 2025년 운영 공고
- 스타트업 심리상담 프로그램도 dcamp 등에서 조달 형태로 발주

### 2-4. 경쟁 대응 전략

> 지금 공백이지만, 공백은 반드시 채워진다. 선점 후 수비 전략을 미리 설계한다.

**위협 시나리오별 대응**

**시나리오 A — 마인드카페·트로스트가 엔터/스포츠 전담 영업팀 신설**

```
예상 시점: 베일러 레퍼런스 케이스 1~2개 공개 후 6~12개월

위협 수준: 중간
  · 그들의 강점: 대기업 HR 채널, 브랜드 인지도, 자본력
  · 그들의 약점: 스포츠/엔터 도메인 깊이 없음, 일반 EAP UI로는
                 4C 체크인·이벤트 캘린더 구현 느림

베일러 대응:
  1. 데이터 선점 — 엔터/스포츠 특화 데이터가 1~2년 이상 쌓이면
     진입 장벽이 됨. 빠른 고객 확보가 최우선.
  2. 도메인 언어 독점 — "컴백 체크인", "경기 전 루틴", "심사 D-7"
     같은 도메인 전용 언어가 범용 EAP가 따라오기 어려운 영역
  3. 코치 네트워크 선점 — 스포츠심리·K팝 도메인 코치 DB를
     경쟁사보다 먼저 확보하면 서비스 품질 격차 유지 가능
```

**시나리오 B — BetterUp 또는 글로벌 플레이어 한국 진출**

```
예상 시점: 2027년 이후 (현재 한국 시장 규모 관심도 낮음)

위협 수준: 낮음~중간
  · 그들의 강점: 글로벌 코칭 네트워크, AI 매칭 기술, 자본력
  · 그들의 약점: 한국어 도메인 데이터 없음, K팝/한국 스포츠
                 문화 이해 부재, 가격 구조 한국 시장 부적합
                 ($100+/session → 한국 기준 고가)

베일러 대응:
  1. 한국어 도메인 특화 — 글로벌 플레이어가 절대 빠르게 복제
     못하는 영역 (PRIPER 심리 프로파일, 한국 스포츠 일정 연동)
  2. 가격 방어선 — 한국 시장 수용 가격대(5~15만원/월)에서 포지션
     유지. 글로벌 플레이어는 이 가격대 대응 불가.
  3. 엔터프라이즈 고착화 — 장기 계약 + 데이터 연속성으로
     전환 비용(switching cost) 높임
```

**시나리오 C — 소속사/구단이 자체 솔루션 개발**

```
예상 시점: 대형 기획사 또는 대형 구단 자체 개발 가능성

위협 수준: 낮음
  · 가능 주체: HYBE, SM, YG 수준의 대형사만 현실적
  · 이들의 한계: 플랫폼 개발·운영 역량 없음, 코치 네트워크 없음,
                타사 대비 데이터 절대적으로 적음
  · 실제 패턴: 대형사는 외부 솔루션 계약을 선호
               (Salesforce, SAP도 직접 만들지 않음)

베일러 대응:
  대형사도 자체 개발 대신 베일러를 선택할 만한 이유를 항상 유지:
  · 플랫폼이 더 정교함 (개인 PRIPER 프로파일 + 7년 종단 데이터)
  · 코치 네트워크를 따로 구축할 필요 없음
  · 위기 감지·라우팅을 직접 설계하면 법적 책임도 직접 짐
```

**베일러의 핵심 방어 자산 (순서대로)**

| 순위 | 방어 자산 | 경쟁사 복제 난이도 |
|---|---|---|
| 1 | 엔터/스포츠 특화 데이터 (시간이 쌓일수록 격차) | 매우 높음 |
| 2 | PRIPER 기반 개인 심리 프로파일 연속성 | 높음 (특허 가능 영역) |
| 3 | 도메인 전문 코치 네트워크 | 중간 (선점 효과) |
| 4 | 4C × 이벤트 캘린더 연동 UX | 중간 (기술 격차 6~12개월) |
| 5 | 레퍼런스 케이스 (첫 고객사 공개 여부) | 낮음 (모방 가능) |

## 3. 규제 및 자격 환경

### 3-1. 현재 법적 상태: "회색지대"

한국은 현재 심리상담 서비스에 대한 **명확한 국가 자격·규제 법령이 없다.**

| 구분 | 현황 |
|---|---|
| 국가 공인 자격 | **없음** (의료인 면허와 별개. 상담심리사, 임상심리사 등은 민간자격 또는 학회 자격) |
| 의료법 적용 여부 | 심리상담 자체는 의료행위 아님 → 의료법 적용 제외 (단, 진단·처방 행위는 의료법 위반 가능) |
| 민간자격 난립 | 검증되지 않은 민간 심리상담사 자격증 수백 개 존재 |
| 온라인 서비스 규제 | 현재 특별한 제한 없음 → 비대면 상담 앱 운영에 법적 장벽 낮음 |

### 3-2. 진행 중인 입법 움직임

- 여야 의원 22명 **'마음건강심리사 및 마음건강상담사 자격에 관한 법률안' 공동발의**
  - 국회 보건복지위원회 심사 중 (2025~2026 내 본회의 상정 예상)
  - 통과 시 심리사·상담사가 국가자격화 → 서비스 제공 요건이 강화될 수 있음

### 3-3. 베일러에 대한 시사점

| 항목 | 판단 |
|---|---|
| 지금 서비스 출시 가능 여부 | **가능** — 법적 장벽 현재 없음 |
| 리스크 요인 | 상담사법 통과 시 자격 요건 소급 적용 가능성 → 협력 상담사 자격 기준을 미리 높게 설정할 것 |
| 권장 대응 | 서비스 내 상담사 자격 기준을 **상담심리사 2급 이상(한국상담심리학회 또는 한국임상심리학회)** 으로 자체 설정 — 법제화 이후에도 기준 충족 가능한 수준 |

## 4. 코칭 포지셔닝 전략

### 심리상담 vs 퍼포먼스 코칭

베일러 B2B를 "심리상담 서비스"가 아닌 **"퍼포먼스 코칭 플랫폼"** 으로 포지셔닝하는 것을 검토.

| 구분 | 심리상담 포지셔닝 | 코칭 포지셔닝 |
|---|---|---|
| 규제 리스크 | 상담사법 입법 시 소급 가능 | **완전 무규제** — 국내외 코칭 면허 요건 없음 |
| 고객 수용성 | "치료 받는다" → stigma 존재 | "루틴/훈련이다" → 자연스럽게 받아들임 |
| 소속사 예산 논리 | 직원 복지 비용 | **퍼포먼스 최적화 투자** |
| 서비스 범위 | 과거/상처 중심 | 현재/목표 중심 |
| 협력 인력 자격 | 상담사 자격 기준 관리 필요 | ICF 등 코칭 자격 (민간, 법적 강제 없음) |

### 권장 구조: 코칭 플랫폼 + 상담 연계

선수/아이돌이 번아웃·불안 등 임상 수준의 이슈를 호소할 경우 순수 코칭 범위를 넘을 수 있음.  
→ **"코칭 세션이 기본, 필요 시 전문 상담사 레퍼럴"** 구조로 양쪽 커버 가능.

## 4-1. 레퍼런스 케이스: A.J. 브라운 사이드라인 독서 (2025년 1월)

### 무슨 일이었나

2025년 1월 NFL 플레이오프(필라델피아 이글스 vs 그린베이 패커스)에서, 이글스 와이드리시버 **A.J. 브라운**이 경기 중 사이드라인 벤치에 앉아 **너덜너덜해진 책을 꺼내 읽는 장면**이 카메라에 잡혀 전세계 바이럴.

- 책: **"Inner Excellence"** (Jim Murphy 저) — 전직 프로야구 선수 출신 퍼포먼스 코치가 쓴 멘탈 루틴 서적
- 브라운은 매 드라이브 사이마다 책을 꺼내 읽으며 집중력을 재설정한다고 설명
- 팀메이트들 사이에서 **"the recipe"** 라고 불릴 만큼 팀 전체가 공유하던 책
- 바이럴 직후 아마존 순위 **523,497위 → 1위** 로 수직 상승

### 베일러에 대한 시사점

1. **멘탈 퍼포먼스 루틴이 가시화됐을 때 stigma가 사라진다** — "심리치료"가 아닌 "경기 전 루틴"으로 프레이밍되면 엘리트 선수도 공개적으로 실천
2. **엘리트 선수들이 이미 이런 도구를 쓰고 있다는 증거** — 수요는 이미 존재, 공식화·디지털화가 안 돼있을 뿐
3. **베일러의 포지셔닝 스토리**: A.J. 브라운이 손으로 하는 멘탈 루틴을 앱으로 체계화한 코칭 플랫폼

> "엘리트 선수의 멘탈 루틴을 디지털로 구조화한다" — 이것이 베일러 B2B의 핵심 메시지가 될 수 있음.

## 4-2. 엔터/스포츠 B2B 기회 요약

### 왜 이 고객이 매력적인가

1. **집단 특성이 명확함**: 소속 선수·아이돌은 공통된 스트레스 패턴 보유 (경기 전후 압박, 데뷔/컴백 주기 스트레스, 대중 노출 부담, 팀 내 인간관계)
2. **B2B 계약 구조**: 개인 결제 설득 불필요, 단가↑, 이탈↓, 예산 집행 안정적
3. **ROI 설명 가능**: 멘탈 관리 → 퍼포먼스·활동 지속성과 직결 → 기업 설득 논리 확보
4. **레퍼런스 파급력**: 엔터/스포츠 업계는 네트워크가 좁아 첫 케이스가 강력한 영업 자산

### 경쟁 공백

- 마인드카페·트로스트는 **일반 직장인 대상 EAP**에 집중
- 아이돌사·스포츠구단 특화 B2B 플레이어: **현재 없음** (2026년 4월 기준 확인)

### 🎯 탑티어 퍼포먼스 코치 관점

**"루틴화가 핵심이다. 위기 개입이 아니라 일상 습관 시스템을 만들어라"**

미국 올림픽위원회(USOC)는 전임 멘탈 퍼포먼스 코치 15명 운영. 업계 표준 프레임:

**4C 멘탈 터프니스 모델**

```
Control    — 감정과 집중력을 통제 가능한 것에만 쏟는 능력
Commitment — 훈련/퍼포먼스 목표에 대한 지속적 헌신
Challenge  — 장애물을 성장 기회로 재프레이밍하는 능력
Confidence — 준비와 경험에서 오는 내적 자신감
```

이 4C를 **앱의 일일 체크인 구조**로 전환:

> "오늘 내가 통제할 수 있는 것에 집중했나?" (Control)  
> "오늘 루틴을 지켰나?" (Commitment)  
> "오늘 어려운 순간을 어떻게 해석했나?" (Challenge)  
> "지금 자신감 수준은?" (Confidence)

**A.J. 브라운의 사이드라인 루틴이 정확히 이 구조** — 매 드라이브 사이 체크인으로 Control/Commitment 리셋.

**ROI 근거 수치 (소속사 설득 논리)**
- 하버드 연구: MLB 선수 중 멘탈 코치와 작업한 선수는 **평균 2.3년 더 길게** 현역 활동
- 심리적 요인이 엘리트 레벨 퍼포먼스 차이의 **최대 30%** 를 설명 (Frontiers in Psychology, 2025)
- 멘탈 터프니스는 선수의 헌신도와 경기 결과 사이의 "bridge" 역할 (2025 연구 확인)

**"K팝 아이돌의 68%가 불안/우울 증상 — 이건 복지가 아니라 리스크 관리다"**

**업계 현실 수치**
- K팝 아이돌 정신건강 증상 보유율: **68%** (한국엔터테인먼트경영협회)

- JYP: 트레이니 대상 **데일리 멘탈 모니터링 앱** 운영 — 알람 수준 시 즉시 개입

심리 전문가 관점의 설계 기준 (Frontiers in Psychology Evidence-Informed Framework 기반):

| 원칙 | 구체적 설계 |
|---|---|
| **3단계 레이어** | 예방(일상 루틴) → 퍼포먼스(코칭 세션) → 임상(상담사 레퍼럴) 명확히 분리 |
| **조기 탐지** | 일일 체크인 패턴으로 번아웃/위기 징후 자동 감지 — 수동 신고 아닌 알고리즘 탐지 |
| **레퍼럴 시스템** | 코칭 범위 초과 시 전문 상담사 즉시 연결 — 경계 명확히 |
| **컨텍스트 특화** | 일반 EAP와 다른 스트레서 인식: 컴백 주기, 팬덤 압박, 소속사 권력관계, 경기 슬럼프 → 이 맥락을 아는 코치 매칭 |
| **익명성 보장** | 개인 세션 내용 소속사 완전 비공개 — 신뢰가 사용률을 결정 |

### 통합 설계 구조

세 관점을 합치면 하나의 아키텍처가 나온다:

```
[Layer 1 — 일상 루틴]
  AI 데일리 체크인 (4C 기반) + 넛지
  → 매일, 경기/컴백 전후 자동 트리거

[Layer 2 — 코칭 세션]
  도메인 특화 코치 AI 매칭
  (스포츠/엔터 컨텍스트 보유 코치)
  → 정기 세션 + 세션 전후 측정

[Layer 3 — 레퍼럴]
  번아웃/위기 패턴 감지 시
  전문 상담사 즉시 연결

[어드민]
  소속사: 집단 트렌드 리포트만 (익명 집계)
  개인 데이터: 완전 비공개
```

**경쟁 우위 포인트**: 마인드카페·트로스트는 일반 직장인 EAP. 베일러가 스포츠/엔터 도메인 컨텍스트(컴백 주기, 경기 사이클, 팬덤 압박)를 앱에 내재화하면 — 이것이 진짜 차별화.

### 🎯 Dr. Sarah Kim | 전 US Olympic Committee 멘탈 퍼포먼스 코치

올림픽 선수들과 일하면서 배운 것 — 최상위 퍼포머일수록 **"나는 멘탈이 약하지 않다"** 는 자의식이 강합니다. "멘탈 관리" 서비스라고 하면 거부 반응이 먼저 옵니다.

```
❌ "멘탈 관리"     →  ✅ "퍼포먼스 루틴"

❌ "상담"          →  ✅ "코치와의 세션"
❌ "번아웃 예방"   →  ✅ "컨디션 피크 유지"
```

| 컴백 2주 전 | 압박 집중 구간 — 체크인 빈도 증가 |
| 쇼케이스/시상식 전날 | 루틴 리마인더 + 코치 즉시 연결 옵션 |
| 활동 종료 후 공백기 | 오히려 이때가 가장 위험 — 자동 웰빙 체크 |

**베일러가 해야 할 한 가지:**
> "소속사 경기 일정/컴백 캘린더와 앱을 연동하세요. 이게 일반 EAP와 완전히 다른 제품이 되는 순간입니다."

캘린더 연동 하나가 베일러를 범용 EAP가 아닌 **도메인 특화 퍼포먼스 플랫폼**으로 만드는 분기점입니다.

---

### 🧬 Dr. James Park | 전 WHO 정신건강 자문, 스포츠심리 전문가

> *"저는 두 가지를 경고하고 싶습니다. 그리고 한 가지 기회를 말씀드리겠습니다."*

**경고 1 — 익명성 설계를 잘못하면 아무도 안 씁니다**

K팝 아이돌이 소속사가 제공한 앱에 솔직하게 입력할까요? 소속사가 "우리가 볼 수 없다"고 해도, 신뢰하지 않으면 형식적으로만 씁니다. 이걸 **"관찰자 효과"** 라고 부릅니다.

해결책은 구조적 분리입니다:

| 원칙 | 구현 방식 |
|---|---|
| 데이터 처리 주체 명시 | 계약서와 앱 모두에서 "데이터 관리자 = 베일러, 소속사 접근 불가" 명시 |
| 계약 해지 후 데이터 소유권 | 소속사가 계약 해지해도 개인 데이터는 유저 소유 |
| 온보딩 첫 화면 고지 | 익명성 보장 원칙을 첫 로그인 시 명시적으로 설명 |

**경고 2 — 코칭과 상담의 경계를 설계에 박아야 합니다**

코칭은 "더 잘하고 싶은 사람"을 위한 것, 상담은 "힘든 사람"을 위한 것입니다. 앱이 이걸 구별 못 하면 위험합니다. 아이돌 68%가 증상 보유 상황에서 — 코칭 세션 중 임상적 개입이 필요한 케이스를 만날 가능성이 높습니다.

**체크인 응답에서 특정 패턴 감지 시 → 코치가 아닌 전문 상담사로 자동 라우팅하는 프로토콜을 MVP에서부터 반드시 포함해야 합니다.**

**기회 — 데이터가 진짜 해자(moat)입니다**

지금 세계에 이런 데이터를 가진 플랫폼은 없습니다:
- "K팝 아이돌의 컴백 주기별 멘탈 패턴 데이터"
- "프로 선수의 경기 전후 컨디션 변화 데이터"

베일러가 이 데이터를 2~3년 쌓으면 — 그게 아무도 복제할 수 없는 자산이 됩니다. 처음부터 데이터 스키마를 이 방향으로 설계하세요. 나중에 바꾸는 건 두 배로 비쌉니다.

## 7. 데이터 Moat 설계 — B2B의 진짜 장기 자산

> "베일러가 이 데이터를 2~3년 쌓으면 — 그게 아무도 복제할 수 없는 자산이 됩니다.  
> 처음부터 데이터 스키마를 이 방향으로 설계하세요. 나중에 바꾸는 건 두 배로 비쌉니다."  
> — Dr. James Park

---

### 7-1. 베일러가 이미 보유한 것 (B2C 기반 자산)

현재 `veilor` 스키마에는 이미 강력한 데이터 기반이 구축되어 있다.

| 테이블 | 수집 중인 데이터 | B2B 활용 가능성 |
|---|---|---|
| `user_signals` | 감정(emotion), 도메인, 점수, 키워드, 신호유형(vent/dig/set/crisis), 임베딩 벡터 | 개인 감정 패턴의 시계열 분석 기반 |
| `crisis_flags` | 위기 트리거 텍스트, 키워드, 심각도, 상태, 해소 시점 | 번아웃/위기 조기 감지 알고리즘 학습 데이터 |
| `persona_instances` | 멀티페르소나 레이어, 일관성 점수, 모순 패턴, 확신도 점수 | 심리 복잡도 프로파일링 |
| `priper_sessions` | 40문항 응답, 4축 점수(ABCD), MSK 가면 코드 12종 | 입사/계약 시점 베이스라인 심리 측정 |
| `user_psych_map_snapshots` | 감정조절/애착안정/소통스타일/경계파워 점수 스냅샷 | 시간에 따른 심리 변화 추적 |
| `pattern_profiles` | 패턴축, 점수, 확신도, 트렌드(상승/하강/유지) | 개인 패턴 변화율 측정 |
| `biometric_data` | 심박수, 스트레스 레벨, 호르몬, 활동 | 신체-심리 연동 분석 |
| `content_consumption` | 완독률, 반응(like/save/skip), 소비 시각 | 콘텐츠 효과 측정 |
| `hypothesis_interventions` | 개입 유형, 텍스트, 우선순위, 필요 위험 레벨 | 어떤 개입이 효과적인지 학습 |

**핵심 강점**: 베일러는 이미 감정 임베딩 벡터(`embedding` 컬럼)를 쌓고 있다. 이는 단순 수치 트래킹이 아닌 **의미 기반 감정 공간 분석**이 가능한 수준.

### 7-2. B2B에서 추가로 수집해야 할 것 (현재 없는 컨텍스트 레이어)

베일러 B2C 데이터에는 **"언제 어떤 외부 이벤트가 있었는가"** 가 빠져 있다. 이게 B2B 데이터 moat의 핵심이다.

#### 추가 필요 테이블 설계안

**① `b2b_org_events` — 소속사 이벤트 캘린더**
```
org_id          uuid        소속사
member_id       uuid        선수/아이돌 (FK → user_profiles)
event_type      text        comeback | showcase | competition | award | hiatus | training_camp | injury
event_name      text        이벤트명
scheduled_at    timestamptz 예정 일시
ended_at        timestamptz 종료 일시
pressure_level  integer     1~5 (소속사가 입력하는 중요도)
outcome         text        win | loss | success | cancelled | postponed (사후 입력)
```

→ 이 테이블이 생기는 순간, `user_signals`의 모든 데이터에 **"컴백 D-14", "경기 D+1", "시상식 당일"** 컨텍스트가 붙는다.

**② `b2b_checkin_sessions` — 타이밍 기반 체크인**
```
member_id       uuid
org_event_id    uuid        FK → b2b_org_events (nullable, 이벤트 연계 체크인)
trigger_type    text        scheduled | event_pre | event_post | slump_auto | manual
checkin_at      timestamptz
4c_control      integer     1~10
4c_commitment   integer     1~10
4c_challenge    integer     1~10
4c_confidence   integer     1~10
free_text       text        선택적 자유기술
routing_result  text        coaching | counseling | self_care (자동 라우팅 결과)
coach_id        uuid        배정된 코치 (nullable)
```

**③ `b2b_coaching_sessions` — 코칭 세션 기록**
```
member_id       uuid
coach_id        uuid
session_at      timestamptz
session_type    text        regular | event_pre | event_post | crisis
pre_4c_scores   jsonb       세션 전 체크인 점수
post_4c_scores  jsonb       세션 후 체크인 점수
coach_notes     text        코치 메모 (멤버 비공개)
session_outcome text        goal_set | pattern_identified | referral_made | routine_reinforced
follow_up_at    timestamptz 다음 세션 예약
```

**④ `b2b_org_aggregate` — 소속사용 익명 집계 (어드민 대시보드 소스)**
```
org_id          uuid
snapshot_date   date
member_count    integer     총 멤버 수
active_rate     float       30일 자발적 사용률
avg_4c_scores   jsonb       {control, commitment, challenge, confidence} 팀 평균
risk_flag_count integer     위험 패턴 감지 건수 (개인 식별 불가)
coaching_sessions_count integer
top_domains     jsonb       가장 많이 나온 감정/도메인 TOP 3
upcoming_event  text        다음 주요 이벤트명 (캘린더 연동)
```

### 7-3. 데이터가 쌓이면 생기는 것 — 시간별 가치 곡선

```
6개월차
  └─ "컴백 2주 전 아이돌의 4C 점수 평균 변화" 패턴 도출 가능
  └─ "어떤 체크인 트리거가 자발적 사용률을 높이는가" 학습 시작

1년차
  └─ "MSK 가면 유형별 번아웃 취약 타이밍" 예측 모델 초안
  └─ "코칭 세션 후 경기/퍼포먼스 결과 상관관계" 분석 가능
  └─ 업계 최초 "K팝 컴백 주기 × 멘탈 패턴" 데이터셋 보유

2~3년차
  └─ 도메인 특화 AI 코치 파인튜닝 가능 (일반 LLM과 완전히 다른 답변)
  └─ "이 프로파일의 선수는 D-7에 루틴 X가 효과적" 개인화 예측
  └─ 이 데이터를 가진 플랫폼은 베일러 뿐 — 복제 불가
```

### 7-4. 베일러 B2C 데이터와 B2B 데이터의 결합 구조

```
[B2C 데이터 — 개인 심리 레이어]          [B2B 데이터 — 외부 이벤트 레이어]
user_signals (감정/도메인/점수)    ←→    b2b_org_events (컴백/경기/시상식)
priper_sessions (MSK 가면 코드)    ←→    b2b_checkin_sessions (4C 타이밍 체크인)
persona_instances (심리 복잡도)    ←→    b2b_coaching_sessions (개입 효과)
user_psych_map_snapshots (변화)    ←→    b2b_org_aggregate (팀 트렌드)
                    ↓
        "외부 이벤트 × 내부 심리 상태" 교차 데이터
        → 세계 어디에도 없는 엔터/스포츠 멘탈 데이터셋
```

### 7-5. 데이터 설계 원칙 3가지

**원칙 1 — 개인 식별과 집계를 물리적으로 분리**
- `member_id`는 `org_id`와 직접 조인 불가 구조로 설계
- 소속사 어드민이 접근하는 모든 뷰는 집계만 노출 (개인 행 접근 차단)
- RLS(Row Level Security) 정책: 소속사 role → `b2b_org_aggregate`만 SELECT 가능

**원칙 2 — 이벤트 컨텍스트를 모든 신호에 자동 태깅**
- `user_signals` INSERT 시 트리거로 가장 가까운 `b2b_org_events`를 자동 조회
- `days_to_event`, `days_from_event` 컬럼 자동 계산 후 `meta` jsonb에 저장
- 이 태깅이 없으면 데이터는 맥락 없는 숫자 더미에 불과

**원칙 3 — 계약 해지 후에도 개인 데이터는 유저 소유**
- 소속사 계약 해지 시 `b2b_checkin_sessions`, `b2b_coaching_sessions`의 `org_id` null 처리
- 개인 데이터(`user_signals`, `priper_sessions` 등)는 유저 계정에 그대로 유지
- 이 정책을 온보딩 첫 화면에서 명시 → 신뢰 확보 → 데이터 품질 확보

```
  10세 시작
  ↓ 7년치 데이터 축적

  · 효과적 개입 이력
  · 4C 점수 성장 궤적

         ↓ 데뷔

[데뷔 후 — 베일러 B2B 성인 모드]
  담당 코치가 7년치 베이스라인 데이터 접근
  ↓

  · "MSK-NRC 유형은 팬덤 압박에 특히 취약"

  ↓

```

이 연속성이 베일러 B2B의 **최종 경쟁 우위**입니다.  
트레이니 때부터 시작한 유저는 데뷔 후에 절대 플랫폼을 바꾸지 않습니다.  
7년치 데이터를 두고 다른 앱으로 이동하는 건 처음부터 다시 시작하는 것과 같으니까요.

## 12. 위기 감지 알고리즘 기준

> "위험 패턴 감지 시 자동 라우팅"의 구체적 발동 조건.  
> 개발자가 임의로 정하지 않도록 기준값을 명시.

---

### 12-1. 위험 스코어 계산 방식

체크인 응답에서 아래 세 가지 신호를 조합해 위험 스코어를 산출.

**신호 A — 4C 절대값**

| 조건 | 포인트 |
|---|---|
| 4C 합산 평균 ≤ 3.0 (10점 만점) | +40 |
| 4C 합산 평균 ≤ 5.0 | +20 |
| 개별 항목 중 1개라도 ≤ 2.0 | +20 |

**신호 B — 연속 하락 패턴**

| 조건 | 포인트 |
|---|---|
| 4C 평균 3일 연속 하락 | +20 |
| 4C 평균 5일 연속 하락 | +35 |
| 이벤트 전 기간 중 하락 (D-7 이내) | 포인트 ×1.5 배수 |

**신호 C — 텍스트 키워드 (기존 `crisis_flags` 로직 활용)**

| 조건 | 포인트 |
|---|---|
| 자유 텍스트에 위기 키워드 포함 (LOW tier) | +20 |
| 위기 키워드 포함 (HIGH tier: 자해·소멸 관련) | +60 |

---

### 12-2. 위험 등급 분류

| 등급 | 스코어 범위 | 발동 조건 | 액션 |
|---|---|---|---|
| **NORMAL** | 0~19 | 해당 없음 | 없음 |
| **LOW** | 20~39 | 경미한 하락 패턴 | 코치에게 소프트 알림 — 다음 정기 세션에서 확인 |
| **MEDIUM** | 40~59 | 지속적 하락 or 절대값 낮음 | 코치 즉시 알림 — 24시간 내 연락 의무 발생 |
| **HIGH** | 60+ | 극단적 절대값 or 위기 키워드 | 코치+상담사 동시 알림 — 즉시 개입 / 트레이니: 보호자 알림 추가 |

---

### 12-3. 특수 규칙

**규칙 1 — 이벤트 컨텍스트 보정**
```
이벤트 D-3 이내이거나 D+1 이내인 경우:
  MEDIUM 임계값을 40 → 30으로 낮춤
  (이 시기 하락은 더 민감하게 반응)
```

**규칙 2 — 체크인 미완료 연속 처리**
```
3일 연속 체크인 미완료:
  → 코치에게 LOW 알림 (스코어 계산 불가 상태)
5일 연속 미완료:
  → 코치에게 MEDIUM 알림
  → 트레이니: 보호자 앱에 미완료 알림
```

**규칙 3 — 급격한 단일 하락**
```
전일 대비 4C 평균 -3.0 이상 단발 하락:
  → 스코어와 무관하게 LOW 알림 자동 발동
  (급격한 외부 사건 가능성 대응)
```

**규칙 4 — HIGH 키워드 즉시 처리**
```
텍스트에서 HIGH tier 위기 키워드 감지 시:
  → 스코어 계산 생략, 즉시 HIGH로 분류
  → 앱 화면에 위기상담 전화번호 즉시 노출 (1393)
  → 코치 + 상담사 동시 알림
  → 로그: crisis_flags 테이블 즉시 INSERT
```

| 등급 | 코치 응답 의무 | 미응답 시 |
|---|---|---|
| LOW | 다음 정기 세션 내 | 없음 |
| MEDIUM | 24시간 내 연락 | 베일러 운영팀 알림 |
| HIGH | 2시간 내 연락 | 베일러 운영팀 직접 개입 |

### 12-5. 알고리즘 기준값 임상 근거

> 스코어 수치가 왜 그 숫자인가 — 자의적 설정이 아님을 근거로 설명한다.

**4C 임계값 근거 (신호 A)**

| 기준값 | 근거 |
|---|---|
| 평균 ≤ 5.0 → +20pts | Mental Toughness Questionnaire(MTQ48) 연구: 4C 평균 5.0 이하는 임상집단의 번아웃 전조 구간과 중첩 (Clough et al., 2002; 후속 메타분석 2019) |
| 평균 ≤ 3.0 → +40pts | 엘리트 선수 연구: 4C 평균 3.0 이하는 "경기 기능 손상(performance dysfunction)" 상태와 높은 상관 (Gucciardi et al., 2015) |
| 개별 항목 ≤ 2.0 → +20pts | 단일 축 극단 하락은 전체 평균보다 특정 영역 위기 신호로 더 민감. Control ≤ 2.0은 무력감, Confidence ≤ 2.0은 자기효능감 붕괴 구간 (Bandura, 1997 자기효능감 이론 기반) |

**연속 하락 패턴 근거 (신호 B)**

| 기준값 | 근거 |
|---|---|
| 3일 연속 하락 → +20pts | 기분 장애 조기 감지 연구: 3일 연속 부정적 자기보고는 일시적 스트레스 반응과 지속성 장애를 구분하는 임계점 (Lamers et al., 2020, EMA 기반 연구) |
| 5일 연속 하락 → +35pts | DSM-5 주요우울장애 진단 기준 "거의 매일, 2주 이상"의 조기 감지 단위. 5일은 7일(1주) 임박 전 개입을 위한 선행 경고 시점 |
| 이벤트 D-7 ×1.5 배수 | 스포츠 경기 전 불안 연구: 경기 7일 전부터 상태 불안(state anxiety)이 유의미하게 증가, 이 기간 하락 신호의 심각도 가중은 임상적으로 정당화 (Jones & Swain, 1995; 후속 K팝 적용 연구 부재 → 스포츠 데이터로 대리 적용) |

**키워드 등급 분류 근거 (신호 C)**

```
HIGH tier 키워드 정의 기준:
  · WHO 자살예방 미디어 가이드라인 (2023) — 자해·자살 관련 직접 표현
  · 한국 자살예방협회 위기 언어 목록 (2024 개정판)
  · 임상 적용: 텍스트 한 건이라도 감지 시 즉시 개입 원칙
    (Columbia Suicide Severity Rating Scale의 "ideation with intent" 구간 대응)

LOW tier 키워드 정의 기준:
  · 번아웃, 소진, 포기, 그만두고 싶다 — 지속적 위험 아니나 모니터링 필요
  · PHQ-9 문항 언어("아무것도 하기 싫다", "즐거운 일이 없다") 기반
```

**세그먼트별 임계값 차등 적용 검토**

| 세그먼트 | 현재 기준 | 조정 권고 | 근거 |
|---|---|---|---|
| 성인 선수/아이돌 | 동일 | 동일 적용 가능 | 성인 임상 데이터 기반 |
| 트레이니 (15~17세) | 동일 | MEDIUM 임계값 40 → 35 하향 | 청소년 정서 변동성이 성인보다 크나, 낮은 점수가 더 심각한 의미를 가질 수 있음 (Larson & Ham, 1993) |
| 트레이니 (12~14세) | 동일 | MEDIUM 임계값 40 → 30, HIGH → 50 하향 | 초기 청소년 자기보고 신뢰도 낮음 → 더 민감하게 반응 |
| 기업 직원 | 동일 | 이벤트 배수 제거 검토 | 기업 이벤트(마감 등)는 스포츠 경기만큼 단일 시점 집중도 낮음 |

> **데이터 축적 후 검증 계획**: 6개월 이상 데이터 확보 시 실제 위기 케이스와 스코어를 대조해  
> 기준값 재보정 예정. 초기 설정은 보수적(과감지 방향)으로 운영.

## 13. 코치 수급 및 검증 기준

> "도메인 아는 코치 10명 직접 영입"의 구체적 기준.  
> 코치 없이 플랫폼은 작동하지 않음 — 공급자 설계가 서비스 품질의 전부.

---

### 13-1. 코치 자격 기준

**필수 요건 (AND 조건 — 모두 충족해야 함)**

| 항목 | 기준 |
|---|---|
| 학력 | 심리학·상담학·체육학·코칭학 관련 학사 이상 |
| 자격증 | 아래 중 하나 이상 보유 |
| 경험 | 선수·아이돌·트레이니 대상 코칭/상담 **최소 1년 이상** |
| 언어 | 한국어 필수. 영어 가능자 우대 (글로벌 확장 대비) |

**인정 자격증 목록**

| 자격 | 발급 기관 | 비고 |
|---|---|---|
| 상담심리사 2급 이상 | 한국상담심리학회 | 심리상담 레퍼럴 역할 가능 |
| 임상심리사 2급 이상 | 한국임상심리학회 | |
| ICF ACC 이상 | 국제코칭연맹 | 코칭 전문성 |
| CMPC | 한국스포츠심리학회 | 스포츠 도메인 |
| 청소년상담사 2급 이상 | 한국청소년상담복지개발원 | 트레이니 모드 담당 코치 필수 |

**우대 요건 (가점)**
- 직접 스포츠 선수/연예인 경력 보유
- K팝 기획사·스포츠 구단 근무 경력
- 번아웃·퍼포먼스 심리 분야 논문 또는 연구 경력

### 13-2. 코치 검증 프로세스 (6단계)

```
[1단계] 서류 심사
  · 자격증 사본 + 경력 증명 제출
  · 합격 기준: 필수 요건 전체 충족

[2단계] 도메인 인터뷰 (30분, 화상)
  · 베일러 운영팀 + 도메인 자문 1인
  · 질문: 실제 케이스 기반 대응 방식, 경계 설정 능력, 위기 대응 경험
  · 평가: 도메인 이해도 / 공감 능력 / 한계 인식

[3단계] 실습 체크인 리뷰
  · 베일러 테스트 계정으로 실제 체크인 응답 3개 리뷰 수행
  · 평가: 위험 신호 식별 정확도, 피드백 언어 적절성

[4단계] 트라이얼 코칭 세션 (유료, 1회)
  · 베일러 내부 테스터(자원자)와 실제 세션 진행
  · 세션 후 테스터 평가 + 운영팀 관찰 피드백

[5단계] 온보딩 교육 (필수 이수)
  · 베일러 플랫폼 사용법 (코치 앱)
  · 코칭↔상담 경계 기준 및 라우팅 프로토콜
  · 미성년자 대응 가이드라인
  · 위기 대응 에스컬레이션 절차
  · 개인정보 보호 의무 및 비밀유지 계약

[6단계] 정식 등록 + 3개월 수습
  · 3개월간 세션 후 유저 평점 모니터링 (4.3 미만 시 추가 면담)
  · 3개월 후 정식 코치 전환
```

### 13-3. 코치 매칭 알고리즘 (초기 → 고도화)

**초기 (MVP): 큐레이션 기반**
```
유저 프로파일 입력값:
  · 도메인 (K팝 / 스포츠 종목)
  · PRIPER MSK 가면 코드
  · 언어 선호 (한국어 / 영어)
  · 세션 스타일 선호 (구조적 / 자유로운)

매칭 로직:
  1. 도메인 일치 코치 필터링
  2. MSK 코드 기반 궁합 테이블 조회 (수동 작성, 초기 10쌍)
  3. 상위 2~3명 카드 노출 → 유저 선택
```

**고도화 (6개월 후): 데이터 기반 자동 매칭**
```
추가 학습 데이터:
  · 세션 후 유저 만족도 점수
  · 세션 전후 4C 점수 변화량
  · 코치별 도메인/MSK 매칭 성공률

목표: BetterUp 97% 매칭 정확도 벤치마크
```

### 13-5. 초기 코치 확보 계획 (현실적 수급 로드맵)

> "도메인 전문 코치 10명"이 어디서, 어떻게, 언제까지 확보되는가.

**확보 채널별 전략**

| 채널 | 예상 수급 인원 | 접근 방법 |
|---|---|---|
| 스포츠 심리 전문가 네트워크 | 3~4명 | 한국스포츠심리학회(KSSP) 회원 디렉토리 — 선수·구단 경험자 직접 컨택 |
| K팝/엔터 업계 내부 | 2~3명 | 소속사 전직 매니저·트레이너 중 코칭 자격 보유자, 업계 인맥 통해 소개 |
| 상담심리대학원 네트워크 | 2~3명 | 엔터·스포츠 심리 전공 석사 과정생 또는 수련 중 전문가 (수퍼바이저 조건부) |
| 해외 교포/글로벌 코치 | 1~2명 | 영어권 K팝 팬덤 출신 코치, 미국 스포츠심리 석사 이상 (글로벌 소속사 대비) |

**단계별 목표**

```
서비스 출시 전 (D-60):
  · 도메인 자문 1인 확보 (명예직, 설계 자문 역할)
    → 스포츠심리학회 또는 엔터 업계 경력 15년+ 원로
  · 코치 후보 10명 서류 심사 완료
  · 최종 검증 통과 코치 5명 계약 완료 (파트타임)

서비스 출시 후 3개월:
  · 고객사 1곳 파일럿 기준 5~10명 멤버 커버
    → 코치 2~3명으로 충분 (1인당 최대 15명, 초기는 5~7명으로 여유 운영)
  · 파일럿 만족도 기반 코치 정식 등록 전환

6개월차:
  · 고객사 3곳 기준 최대 90명 → 코치 6~7명 필요
  · 트레이니 전문 코치 별도 1~2명 추가 확보
  · 기업 모듈 코치 풀 분리 시작 (엔터/스포츠와 별도 트랙)
```

**코치 수급 리스크 및 대응**

| 리스크 | 가능성 | 대응 |
|---|---|---|
| 도메인 전문 코치 절대 수 부족 | 중간 | 초기에는 "스포츠 OR 엔터" 경험자로 기준 완화 가능. 단, 도메인 자문과 함께 온보딩 |
| 코치 이탈 (경쟁사 스카우트) | 낮음~중간 | 6개월 전속 계약 + 데이터 기반 성과 보너스 구조로 락인 |
| 자격 기준 충족자 부족 | 낮음 | 한국 내 스포츠심리학 석사 배출 연 200명+ (서울대·고려대·연세대·한국체대). 실무 경험 1년 기준 충족 가능 풀 충분 |
| 파트타임 코치의 응답 SLA 미준수 | 중간 | 계약서에 SLA 명시, 미준수 2회 시 자동 계약 해지 조건 삽입. 백업 코치 상시 대기 |

| 주니어 채용 축소 | 대기업 신입 TO 2023→2026 평균 38% 감소, 경력직 전환 채용 확대 |
| EAP 도입 논의 | 2026년 300인 이상 기업 EAP 의무화 논의 (고용노동부 가이드라인 발표 예정) |
| 번아웃 호소 비율 | 2026년 직장인 번아웃 65% (MZ세대 78%) |
| 정신건강 서비스 이용률 | 번아웃 경험자 중 실제 상담 이용 11% — 89%는 받고 싶어도 안 감 |

**기업 직원이 상담을 안 가는 이유 (2026 서베이 기반)**
```
1위 (44%) — "약하다는 인식 받을까봐" (낙인 효과)
2위 (31%) — "회사에 알려질까봐" (비밀유지 불신)
3위 (15%) — "바빠서 시간이 없어서"
4위 (10%) — "어디서 어떻게 받는지 몰라서"
```

→ 스포츠 선수·아이돌과 **완전히 동일한 구조**. 포지셔닝 해법도 동일.

### 14-5. 기업 버전 4C 체크인 언어 설계

> 동일한 4C 모델, 기업 맥락 언어로 재표현

| 4C 축 | 스포츠/엔터 표현 | 기업 표현 |
|---|---|---|
| Control | "내 감정과 루틴을 내가 조절하고 있다" | "내 업무 흐름과 감정을 내가 통제하고 있다" |
| Commitment | "이 목표를 향한 의지가 흔들리지 않는다" | "내 커리어 방향에 대한 확신이 있다" |
| Challenge | "어려운 상황을 성장 기회로 본다" | "압박 상황을 도전으로 받아들이고 있다" |
| Confidence | "내 실력과 판단을 신뢰한다" | "내 역량과 성과를 스스로 인정하고 있다" |

**기업 버전 체크인 UX 예시**
```
오늘 업무 시작 전, 30초 루틴 ―

지금 이 순간,
[업무 흐름을 내가 통제하고 있다] 1 ─────● 10

이번 주 커리어 방향에 대한 확신
[흔들린다] 1 ─────● 10 [확고하다]

지금 나에게 어울리는 한 문장을 고르세요.
  ○ 오늘도 해낼 수 있다
  ○ 좀 쉬고 싶다
  ○ 모르겠다
```

### 15-1. 진입 순서 및 근거

```
  · 코치 세션 만족도 4.0/5.0+
  · 4C 평균 점수 변화 (입력 대비 출력)
  · 소속사 담당자 대시보드 주간 방문율

수익:
  · 파일럿 요금 (정가 50%) 적용
  · 코칭 세션 수 기반 실비 정산
```

**Phase 2 — 엔터 진입 + 트레이니 모듈 출시 (D+90 ~ D+180)**

```
목표: 엔터 고객사 1~2곳 추가. 트레이니 모드 출시.

추가 개발:
  · 트레이니 전용 UI (연령대별: 9~11 / 12~14 / 15~17)
  · 보호자 앱 (알림 수신 + 동의 플로우)
  · 방출 케어 프로토콜 (30일 플로우)
  · K팝 이벤트 캘린더 (컴백 / 심사 / 활동 중단)

운영:
  · 트레이니 전문 코치 1~2명 추가 계약
  · 미성년자 보호 가이드라인 내부 확정
  · 스포츠 파일럿 케이스 스터디 작성 (레퍼런스 영업 활용)

검증 지표:
  · 트레이니 보호자 동의율 80%+
  · 트레이니 체크인 이행률 50%+ (성인 대비 낮아도 정상)
  · 소속사 "트레이니 풀" 플랜 업셀 전환율
```

**Phase 3 — 스포츠+엔터 스케일업 (D+180 ~ D+365)**

```
목표: 고객사 5곳+ / 멤버 수 100명+ 달성

초점:
  · 코치 풀 확대 (6~8명)
  · 어드민 대시보드 고도화 (임원 브리핑 리포트 자동화)
  · 코치 매칭 알고리즘 데이터 기반 고도화 시작
  · 엔터프라이즈 플랜 첫 계약 (51명+ 고객사)
  · 상담사법 입법 동향 모니터링 + 필요시 자격 기준 업데이트

수익 목표:
  · ARR(연간 반복수익) 3억원 달성
    (예: 고객사 5곳 × 평균 50명 × 69,000원 × 12개월 = 2억 700만원
     + 애드온 세션 수익 + 파일럿 졸업 정가 전환)
```

**Phase 4 — 기업 모듈 진입 (D+365 이후)**

```
조건: 스포츠·엔터 레퍼런스 3곳 이상, 위기 감지 알고리즘 데이터 검증 완료,
      기업 전문 코치 풀 확보

추가 개발:
  · 기업 버전 언어 레이어 (섹션 14 기준)
  · 기업 이벤트 캘린더 (QUARTERLY_CLOSE, HR_REVIEW 등)
  · 슬랙·Teams 연동 (기업 업무 환경 통합)
  · HR 어드민 대시보드 (섹션 14-7 기준)

영업 채널:
  · HR 컨설팅사 파트너십 (간접 영업)
  · 기업 복지 플랫폼 연동 (카카오페이 복지포인트 등)
  · 대기업 HR 담당자 직접 영업 (스포츠·엔터 레퍼런스 활용)
```

## 16. 법적·계약 설계

> 서비스 운영의 법적 기반. 고객사·코치·유저 세 주체 간 계약 구조를 명확히 한다.

---

### 16-1. 계약 주체 구조

```
베일러 ←→ 고객사 (소속사/구단/기업)
  · B2B 서비스 이용 계약
  · 개인정보 처리 위탁 계약 (필수)
  · 비밀유지계약 (NDA)

베일러 ←→ 코치
  · 코치 업무위탁 계약 (도급 또는 위임)
  · 비밀유지계약 (세션 내용 절대 외부 공유 금지)
  · SLA 계약 (응답 의무 명시)

고객사 ←→ 멤버 (선수/아이돌/직원)
  · 베일러 서비스 이용 동의 (개인정보 수집·이용)
  · 트레이니(미성년): 보호자 동의 추가 필수
  · "소속사는 개인 세션 내용을 볼 수 없다" 명시
```

---

### 16-2. 표준계약서 핵심 조항 (고객사-베일러)

**1. 서비스 범위**
```
베일러는 다음을 제공한다:
  · 멘탈 퍼포먼스 체크인 플랫폼 접근권 (SaaS)
  · 퍼포먼스 코칭 세션 연결 (플랜 할당량 범위 내)
  · 집계 데이터 리포트 (비식별화 팀 단위)
  · 위기 감지 알림 (등급별 코치 통보)

베일러는 다음을 제공하지 않는다:
  · 의료·심리치료 서비스
  · 개인 세션 내용 소속사 공개
  · 특정 멤버에 대한 진단·판정
```

**2. 개인정보 처리**
```
· 베일러는 개인정보보호법에 따라 개인정보 처리수탁자로서 의무를 진다
· 고객사는 소속 멤버의 개인정보 수집·처리에 대한 동의를 확보할 의무를 진다
· 계약 종료 시 고객사의 법인 연결 데이터는 null 처리,
  개인 PRIPER 데이터는 개인 소유 유지 (베일러 B2C 전환 선택권 부여)
· 데이터 보관 기간: 계약 종료 후 1년 (법정 의무 기간 준수)
```

**3. 책임 한계**
```
베일러의 책임:
  · 위기 감지 알고리즘 정상 작동 의무
  · HIGH 등급 감지 시 코치 2시간 내 통보 의무
  · 플랫폼 가용성 99% SLA

베일러의 면책:
  · 코치 세션 내용의 효과에 대한 보증 없음
  · 의료적 진단·치료 결과에 대한 책임 없음
  · 위기 감지 후 코치/고객사의 미대응으로 발생한 사건
```

**4. 계약 해지 및 데이터 처리**
```
· 고객사의 중도 해지: 잔여 월 요금 환불 없음 (월 단위 계약)
· 베일러의 서비스 중단: 30일 전 사전 통보 + 데이터 이관 지원
· 해지 후 30일 내 데이터 다운로드 요청 가능 (집계 데이터에 한함)
```

---

### 16-3. 코치 업무위탁 계약 핵심 조항

```
1. 업무 범위
   · 베일러 플랫폼을 통한 퍼포먼스 코칭 세션 제공
   · 위기 알림 수신 및 SLA 내 응답
   · 월 1회 슈퍼비전 참여

2. 비밀유지
   · 세션 내용 및 유저 정보 외부 공유 절대 금지
   · 위반 시: 계약 즉시 해지 + 손해배상 청구 가능

3. SLA
   · HIGH 등급: 2시간 내 응답 (24시간 365일)
   · MEDIUM 등급: 24시간 내 응답
   · 2회 연속 미준수 시 계약 해지

4. 수수료
   · 세션당 35,000원 (베일러 플랫폼 통해 정산)
   · 월 정산 (익월 10일)
   · 부가세별도

5. 전속 조항
   · 베일러 고객사 멤버에 대한 개인 계약 금지
     (베일러 외부에서 동일 멤버와 코칭 계약 체결 시 즉시 해지)
```

---

### 16-4. 미성년 트레이니 보호자 동의 구조

```
법적 근거:
  · 개인정보보호법 제22조: 만 14세 미만 아동 법정대리인 동의 필수
  · 아동복지법: 아동 심리·정서 서비스 제공 시 보호자 권익 보호

동의 플로우:
  [만 14세 미만]
    보호자가 직접 앱 설치 → 보호자 인증(본인인증) →
    자녀 프로필 생성 → 서비스 범위 안내 + 동의 →
    자녀 앱 초대 링크 발송

  [만 14~17세]
    본인 앱 설치 → 본인 기본 동의 →
    보호자 이메일 자동 발송 (동의 확인 링크) →
    보호자 동의 완료 후 서비스 활성화

보호자에게 공개되는 정보:
  · 체크인 완료 여부 (날짜/시간만)
  · 위기 등급 알림 (MEDIUM 이상 시)
  · 코치 세션 예약 일정

보호자에게 비공개:
  · 체크인 상세 점수
  · 코치 세션 내용
  · 자유 텍스트 입력
```

### 17-3. 자주 받는 질문 (예상 Q&A)

**Q. 선수/아이돌이 실제로 쓸까요?**
> A. "상담 받으러 간다"가 아니라 "오늘 컨디션 체크"입니다.
> 30초, 익명성 보장, 자발성 강조. 해외 BetterUp 데이터: 코칭 도입 기업 참여율 83%.
> 소속사는 개인 점수를 볼 수 없다는 사실을 멤버에게 명확히 고지합니다.

**Q. 소속사 담당자가 개인 상태를 볼 수 있나요?**
> A. 절대 안 됩니다. 기술적으로 차단되어 있습니다 (Row Level Security).
> 담당자 화면에는 "팀 평균"과 "주의 인원 수(익명)"만 표시됩니다.
> 이 구조를 계약서에 명시하고, 위반 시 베일러가 법적 책임을 집니다.

**Q. 코치는 어떤 분들인가요?**
> A. 스포츠 심리 전문가, 퍼포먼스 코칭 자격 보유자, 해당 도메인 경력자.
> 베일러 자체 심사 + 슈퍼비전 구조로 품질을 관리합니다.
> 코치 세션 내용은 소속사에 일절 공유되지 않습니다.

**Q. 위기 상황 발생 시 어떻게 되나요?**
> A. HIGH 등급 감지 시 → 담당 코치 2시간 내 알림 → 코치가 직접 연락.
> 동시에 유저 화면에 자살예방상담전화(1393) 즉시 노출.
> 트레이니는 보호자에게도 즉시 알림 발송.
> 소속사 담당자에게는 "위기 인원 1명 감지, 코치 대응 중"만 전달 (익명).

**Q. 계약 후 얼마나 걸리나요?**
> A. 계약 다음날 조직 세팅 완료.
> 멤버 초대 → 온보딩 → 첫 체크인까지 3~5일.
> 담당자 대시보드는 체크인 데이터 누적 후 7일째부터 의미 있는 리포트 제공.

**Q. 파일럿 조건이 있나요?**
> A. 3개월 파일럿 가능. 정가의 50% 적용.
> 파일럿 종료 후 4C 점수 변화 + 코칭 세션 만족도 데이터 제공.
> 효과가 없으면 연장 계약 없이 종료 가능.
