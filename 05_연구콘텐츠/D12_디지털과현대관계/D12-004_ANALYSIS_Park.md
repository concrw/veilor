## 1. 이론 평가

"관계의 공연화(performativity of relationship)" 개념은 SNS가 관계 경험에 미치는 구조적 영향을 포착하는 가장 정확한 이론적 틀이다. 어빙 고프만의 "자아 표현의 연극학(dramaturgy)"을 디지털 관계 맥락에 적용한 이 테제는, SNS가 단순한 소통 도구가 아니라 관계 자체를 재구성하는 무대 장치임을 명확히 한다.

과도한 SNS 관계 전시가 내부 불안정을 외부 인정으로 보완하는 역설적 패턴이라는 주장은 임상적으로 중요하다. SNS 과시와 관계 불안의 역관계는 표면적 관찰과 반직관적이기 때문에, 플랫폼 설계자와 일반 사용자 모두가 이 패턴을 오해하기 쉽다.

SNS 공개 타이밍 불일치가 관계 진지도 인식 차이로 발전한다는 분석은 현대 커플 상담에서 실제로 가장 빈번하게 등장하는 갈등 원인 중 하나를 정확히 짚었다. 공개=약정 공식화 기능은 디지털 이전 시대의 "가족 소개" 관계와 기능적으로 동일한 역할을 수행한다.

---

## 2. 선행 연구 검토

Festinger(1954)의 사회 비교 이론은 SNS 비교 심리의 직접적 이론 기반이다. Chou & Edge(2012)는 Facebook 사용과 "다른 사람들은 나보다 행복하다" 인식의 상관관계를 실증했다. Hanna et al.(2017)은 파트너 인스타그램 사용이 관계 불만족과 연결되는 패턴을 분석했다.

M43 이론의 기여는 이 연구들을 MSK 가면 유형별로 차별화하여 분석한다는 점이다. "SNS가 관계에 나쁘다"는 일반론이 아니라, "어떤 MSK 가면 유형의 사람이, 어떤 SNS 행동에서, 어떤 관계 영향을 경험하는가"로 예측 모델을 정밀화한다.

한계: TikTok·YouTube Shorts 등 단편 동영상 플랫폼이 관계 공연화에 미치는 영향이 Instagram·Facebook 중심 분석에서 충분히 다루어지지 않는다. 숏폼 콘텐츠 플랫폼에서의 커플 콘텐츠 생산은 공연화의 강도를 크게 높이며, 이는 별도의 연구 영역으로 발전해야 한다.

## 3. 실제 케이스 분석

**케이스 A: 커플 인플루언서의 관계 붕괴**
팔로워 수십만 명의 커플 인플루언서가 SNS에서 완벽한 관계를 공연하면서 실제 관계는 심각한 소통 단절을 경험한 사례. 외부 인정(좋아요·댓글·팔로워)이 관계 만족도의 실질적 측정 지표를 대체하게 된 구조적 함정. 관계 전시가 수익 모델과 결합될 때 공연화가 극도로 강화되는 특수 케이스.

**케이스 B: SNS 공개 타이밍 갈등**
한 파트너는 3회 만남 후 "공식 커플"로 SNS 공개를 원했고, 다른 파트너는 6개월 이상 관계를 확인한 후 공개하고 싶었던 갈등 사례. 공개 타이밍 차이가 단순히 습관 차이가 아니라 관계에 대한 진지도와 미래 계획의 불일치를 반영한다는 것을 두 파트너 모두 인식하지 못했다. 명시적 대화 없이 갈등이 지속되다 관계 종료.

**케이스 C: HAB 차이와 SNS 사용 기대 갈등**
HH 파트너(SNS 비공개 원칙, 프라이버시 강조)와 LL 파트너(가족·지인 관계에서 SNS 활동이 관계 참여의 증거)의 갈등. 이 케이스는 SNS 사용 패턴 차이가 단순 취향이 아니라 HAB 문화 배경에서 형성된 관계 언어의 차이임을 보여준다.

## 4. 보완 제안

**이론 확장 1: 플랫폼별 공연화 강도 차이**
모든 SNS 플랫폼이 동일한 수준의 관계 공연화를 유발하지 않는다. LinkedIn(전문 정체성), Instagram(라이프스타일 큐레이션), BeReal(반-공연화 설계), TikTok(알고리즘 기반 확산) 각각의 관계 공연화 메커니즘과 강도를 분류하는 플랫폼별 공연화 지수(PPI) 모델이 필요하다.

**이론 확장 2: SNS 단절 실험과 관계 변화**
"SNS 30일 끊기" 실험 참여자들의 관계 만족도 변화를 추적한 연구에서, 단절이 관계를 개선한 사례와 오히려 파트너의 불안을 높인 사례(SNS 활동을 통해 파트너 안부를 확인하던 패턴이 차단)가 공존한다는 발견을 이론에 통합해야 한다.

**방법론 제안**: 커플 단위 SNS 사용 패턴 일치도 측정 + 관계 만족도 상관 연구. 개인 단위 분석을 넘어 커플 상호작용 단위 분석이 필요하다.

## 5. Veilrum 적용 방안

**5-1. 반-공연화 플랫폼 철학**
- 외부 SNS 연동 완전 금지: Veilrum 프로필이 Instagram/TikTok 하이라이트 릴이 되는 것을 구조적으로 방지
- "지금 이 순간" 비공개 감정 체크인: 공개 포스팅이 아닌 파트너에게만 전달되는 실시간 감정 공유 공간
- 좋아요 수치 제거: 관계 연결의 외부 인정 의존 구조를 설계에서 제거

**5-2. 진정성 연결 도구**
- SNS 비교 심리 인식 워크숍: "당신이 보는 다른 커플의 SNS가 실제가 아니다"를 M43 연구 기반으로 교육
- 관계 공개 협상 가이드: SNS 공개 타이밍, 범위, 방식에 대한 커플 대화 구조화 도구 — "공개 의미 탐색 대화"
- HAB 기반 SNS 기대치 교육: 문화 배경에 따른 SNS 관계 언어 차이 이해

**5-3. 관계 공연화 감지 도구**
- 사용자 SNS 관계 게시 빈도 + MSK 가면 패턴 연계 인사이트: "당신의 SNS 관계 게시가 최근 증가했습니다. 관계 내 어떤 감정을 경험하고 있나요?" 비판단적 체크인
