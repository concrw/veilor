## 1. 이론적 배경

존 볼비는 초기 애착 경험으로부터 형성된 내적 작동 모델(Internal Working Model, IWM)이 이후 모든 친밀 관계의 청사진이 된다고 주장했다. IWM은 자기(self)와 타인(other)에 대한 인지적-감정적 표상 체계로, "나는 사랑받을 가치가 있는가", "타인은 신뢰할 수 있는가"라는 핵심 질문에 대한 암묵적 답변을 포함한다.

M43 D2 디비전은 IWM이 HAB(습관화 프레임워크) 층위에서 관계 자동화의 심층 기반이 되고, MSK(Multi-Self Kaleidoscope) 층위에서는 자기 표현 방식의 근본 구조를 형성한다는 이중 연결을 탐구한다.

---

## 2. 핵심 명제

> **명제**: 초기 애착을 통해 형성된 IWM은 성인 관계에서 자기충족적 예언으로 작동하며, 교정적 경험은 이 IWM을 재조직화함으로써 관계 패턴의 근본적 변화를 가능하게 한다.

## 3. RQ별 분석

### RQ1. 초기 애착 유형이 성인 연애 방식에 미치는 직접 영향은?

성인 애착 연구(Hazan & Shaver, 1987; Bartholomew & Horowitz, 1991)는 초기 애착 유형이 다음 영역에서 직접적 영향을 미침을 보여준다:

**파트너 선택**:
- 불안형: 강렬한 감정적 반응을 유발하는 파트너 선택 경향 (종종 회피형과 쌍을 이룸)
- 회피형: 독립적이고 덜 요구적인 파트너 선택, 그러나 파트너가 거리를 둘 때만 안심
- 혼란형: 매력과 위협을 동시에 가진 파트너에게 끌림 (과거 학대자의 특성과 유사)

**갈등 방식**:
- 안정형: 갈등을 문제 해결 기회로 봄, 수용적이고 협력적
- 불안형: 갈등을 관계 위협으로 지각, 증폭하거나 과도하게 달래려 함
- 회피형: 갈등 최소화 또는 철수, "큰 문제 없음" 프레임
- 혼란형: 갈등 시 혼란스럽고 모순적인 반응

**성적 친밀감**:
- 안정형: 성을 친밀감 표현의 일부로 통합
- 불안형: 성을 관계 확인의 수단으로 사용, 불안 감소 목적
- 회피형: 성적 친밀감과 정서적 친밀감을 분리

### RQ2. 초기 애착 손상을 성인기에 교정하는 경험의 유형과 조건은?

교정적 경험(Corrective Experience)의 유형 분류:

**유형 1: 안정형 파트너십**
- 오랜 기간 안정 애착형 파트너와의 관계
- 조건: 최소 2-3년의 일관된 관계, 파트너의 감정적 가용성
- 한계: 자신의 패턴이 파트너를 지치게 하거나 관계를 파괴할 위험

**유형 2: 장기 심리치료**
- 치료사와의 안전한 애착 관계 경험
- 조건: 치료사의 지속적 가용성, 내담자의 성찰 동기, 최소 1년 이상
- 효과: IWM의 명시적 재조직화 가능

**유형 3: 부모 경험**
- 자신의 자녀와의 관계에서 교정적 경험 발생
- 메커니즘: 자녀를 돌보는 과정에서 자신도 안전 기지를 경험
- 위험: 자신의 미해결 애착 패턴을 자녀에게 전이할 가능성

**유형 4: 영적/공동체 소속**
- 종교 공동체, 치유 집단, 깊은 우정 네트워크
- 조건: 집단의 심리적 안전성과 일관성

### RQ3. 내적 작동 모델(IWM)은 어떻게 관계 기대치를 형성하는가?

IWM은 세 가지 층위에서 관계 기대치를 형성한다:

**인지적 층위 (Cognitive Level)**:
- "파트너는 결국 나를 떠날 것이다" (불안형의 버림받음 스키마)
- "친밀해지면 나는 통제당할 것이다" (회피형의 침범 스키마)
- 이 기대치는 선택적 주의(selective attention)를 통해 확증 증거를 수집한다

**감정적 층위 (Emotional Level)**:
- IWM에 불일치하는 파트너 행동은 "이상하다"는 감각을 유발
- 불안형은 파트너가 안정적으로 반응할 때 불안해지는 역설 경험
- 회피형은 파트너가 가까이 올 때 불편감 자동 발생

**행동적 층위 (Behavioral Level)**:
- HAB 프레임워크에서 IWM은 관계 행동의 기본값(default setting)으로 작동
- MSK에서는 IWM에 기반한 가면이 자동으로 착용됨
- 자기충족적 예언 메커니즘: 불안형의 집착 행동이 파트너를 실제로 떠나게 만듦

### RQ4. 초기 애착과 현재 관계 만족도 간 매개 변수는?

메타분석 연구들이 확인한 주요 매개 변수:

1. **성찰 기능(Reflective Functioning, RF)**: 초기 애착 손상이 있더라도 RF가 높으면 관계 만족도 영향이 완충됨
2. **정서 조절 전략**: 초기 애착 → 정서 조절 전략 형성 → 갈등 해결 능력 → 관계 만족도
3. **내현적 자존감**: 초기 안정 애착 → 높은 자존감 → 관계에서 자기주장 → 만족도
4. **사회적 지지 활용 능력**: 초기 안정 애착은 타인의 지지를 수용하는 능력을 발달시킴
5. **의사소통 방식**: IWM이 직접 소통 vs. 간접/회피적 소통 패턴을 결정

### RQ5. 초기 애착 유형이 Veilrum 사용 패턴에 반영되는가?

Veilrum 사용 데이터 분석 가설:

- **불안형**: 알림 설정이 과도하게 민감, 응답 대기 시간 짧음, PRIME 지수 변동 폭 큼. codetalk 사용에서 관계 확인 욕구 반영 표현 빈발.
- **회피형**: DIVE 세션 이탈률 높음, 특히 감정 심화 구간. 프로필 설정에서 개인 정보 최소 공개. 파트너 연결 기능 사용률 낮음.
- **혼란형**: 사용 패턴 불규칙, 갑작스러운 탈퇴 후 재가입 반복. 위기 개입 알고리즘 트리거 빈도 높음.
- **안정형**: 정기적이고 일관된 사용, DIVE 세션 완료율 높음, 커뮤니티 기여도 높음.

Veilrum의 행동 데이터 분석 시스템은 이 패턴을 감지하여 맞춤형 지원을 제공한다.

## 4. IWM-MSK-HAB 삼중 연결 모델

```
[초기 애착 경험]
    ↓ 내면화
[IWM 형성]
    ├── 자기 표상 (나는 사랑받을 가치가 있다/없다)
    └── 타인 표상 (타인은 신뢰할 수 있다/없다)
         ↓
    MSK: 기본 가면 구조 결정
    HAB: 관계 자동화 패턴 설정
         ↓
[성인 관계 패턴]
    → 자기충족적 예언
    → 교정 경험에 의한 IWM 재조직화 가능
```

## 5. Veilrum 연결: PRIME - 관계 히스토리 분석

PRIME 관계 히스토리 분석 기능:

1. **생애 관계 타임라인**: 초기 양육자 관계부터 현재까지의 주요 관계를 시각적으로 정리
2. **애착 패턴 추출**: 각 관계에서 반복되는 패턴(상대 유형, 갈등 방식, 종료 방식)을 자동 분류
3. **IWM 지도 생성**: 수집된 데이터를 기반으로 개인의 IWM 프로파일(자기/타인 표상) 시각화
4. **교정 경험 식별**: 히스토리 중 IWM에 변화를 준 관계 경험을 하이라이트하고 분석
5. **현재 관계 연결**: 히스토리 패턴과 현재 관계 패턴의 유사점/차이점 분석 → 인사이트 제공
6. **성장 추적**: 시간에 따른 관계 패턴 변화를 시각화하여 교정 여정 지원

---

## 6. 결론

초기 애착은 성인 관계의 청사진을 제공하지만, 이 청사진은 불변하지 않는다. IWM의 재조직화는 교정적 경험, 치료적 관계, 의도적 성찰을 통해 가능하며, HAB 프레임워크는 이 변화가 어떻게 새로운 습관으로 자동화되는지를 설명한다. MSK는 IWM 변화가 자기 표현 방식의 유연성 증가로 이어지는 과정을 추적한다. Veilrum의 PRIME 관계 히스토리 분석은 이 여정을 데이터 기반으로 지원하는 강력한 도구다.
