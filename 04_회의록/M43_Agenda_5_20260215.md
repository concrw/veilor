## 데이터 수집 방법론

### Track 1: 정량 데이터 (설문)

**일간 로그 (Daily Log)**

**시간:** 매일 저녁 (1분)

**항목:**
- 오늘 성적 활동 여부 (Y/N)
- 파트너 (선택 리스트)
- 만족도 (1-10)
- 특이사항 (선택)

**도구:** 모바일 앱 (푸시 알림)

**목표 응답률:** 90%+

---

**주간 설문 (Weekly Survey)**

**시간:** 매주 일요일 (10분)

**항목 (20문):**
- 관계 상태 (5문)
  - 친밀감, 갈등, 소통, 만족도, 변화
- 감정 상태 (5문)
  - 행복도, 불안, 스트레스, 외로움, 흥분
- 성적 활동 (5문)
  - 빈도, 다양성, 만족도, 욕구, 문제
- 사회적 활동 (5문)
  - 교류, 이벤트, 갈등, 지원, 고립

**척도:** 7점 Likert

**목표 응답률:** 85%+

---

**월간 평가 (Monthly Assessment)**

**시간:** 매월 마지막 주 (30분)

**7개 프레임워크 평가:**

**1. MSK (Mask Types) - 5분**
- 12종 가면 중 현재 주요 가면
- 가면 전환 빈도
- 가면 간 갈등

**2. HAB (Habitus) - 5분**
- 현재 Habitus 유형 (HH/HL/LH/LL)
- 파트너와의 Habitus 조합
- 계급 이동 경험

**3. V/H (Villain-Hero) - 5분**
- 관계 궤적 현재 위치
- Villain/Hero 역할
- 전환점 경험

**4. SEX (Sexual Spectrum) - 5분**
- 주도/수용 축
- 빠름/느림 축
- 시각/촉각/청각 축

**5. M/S (Money-Sex) - 3분**
- 거래적 요소 인식
- 경제적 역학
- 불균형 경험

**6. DIV (Divorce Types) - 3분**
- 해체 가능성
- 해체 경로 예측
- 재결합 의향

**7. CHG (Change Readiness) - 4분**
- 변화 준비도
- 변화 시도
- 저항 요인

**목표 응답률:** 80%+

---

**분기 종합 평가 (Quarterly Comprehensive)**

**시간:** 분기 마지막 주 (1시간)

**내용:**
- 3개월 회고
- 주요 관계 변화
- 인사이트 정리
- 향후 3개월 전망
- 7개 프레임워크 종합

**형식:** 반구조화 온라인 설문

**목표 응답률:** 75%+

---

### Track 2: 정성 데이터 (인터뷰)

**월간 1:1 인터뷰 (Monthly Interview)**

**빈도:** 월 1회  
**시간:** 1시간  
**면접관:** Team Lead  
**참여자:** 각 Specialist

**구조:**

**도입 (10분):**
- 라포 형성
- 최근 상태 확인
- 주요 이슈 파악

**심화 탐색 (40분):**
- 해당 도메인 경험
- 구체적 사례
- 감정/생각 탐색
- 인사이트 도출

**프레임워크 적용 (10분):**
- 7개 프레임워크 관점에서 재해석
- 패턴 확인
- 이론 검증

**녹음:** 동의 시 (익명화)  
**전사:** AI 자동 + 수동 검증

**목표 완료율:** 90%+

---

**분기 그룹 인터뷰 (Quarterly Focus Group)**

**빈도:** 분기 1회  
**시간:** 2시간  
**참여자:** Division별 (평균 19명)  
**진행:** Division Head

**목적:**
- 도메인 간 교차 인사이트
- 집단 역학 관찰
- 공동 패턴 발견
- 지식 공유

**주제 예시:**
- D5: "BDSM과 바닐라의 만족도 차이"
- D9: "퀴어 관계의 고유한 역학"
- D7: "폭력 생존자의 회복 경로"

**녹화:** 동의 시  
**분석:** 주제 분석 (Thematic Analysis)

**목표 참여율:** 80%+

---

### Track 3: 영상 데이터 (드론 촬영)

**동의자만 (예상 60-70%)**

**촬영 프로토콜:**

**사전 준비:**
- 24시간 전 신청 (앱)
- 파트너 동의 확인
- 촬영 룸 예약
- 드론 설정 확인

**당일 절차:**
1. 아침 재확인 (푸시 알림)
2. 룸 도착 30분 전 최종 확인
3. 드론 작동 확인
4. 세션 시작
5. 중단 가능 (언제든)

**촬영 내용:**
- 성적 활동
- 관계 상호작용
- 비언어적 소통
- 신체 반응

**촬영 각도:**
- 고정 드론: 전체 뷰
- 이동 드론: 클로즈업
- 자동 추적: 얼굴/신체

**빈도:** 주 1-2회 (선택)

---

**후처리:**

**즉시 (촬영 후 1시간):**
- 얼굴 자동 모자이크
- 음성 변조 (선택)
- 식별 정보 제거

**24시간 내:**
- 참여자 영상 확인 (앱)
- 삭제 요청 가능
- 승인 시 암호화 저장

**분석:**
- 행동 패턴
- 상호작용 역학
- 감정 표현
- AI 자동 분석 (동작 인식)

**보관:**
- 암호화 저장
- 5년 (삭제 요청 시 즉시)

## 동의 프로세스

### 2단계 동의 체계

**Stage 1: 포괄 동의 (채용 시)**

**4종 동의서:**
- A. 성적/관계 데이터 공유 (필수)
- B. 드론 촬영 (선택)
- C. 비밀 유지 서약 (필수)
- D. 피임 의무 (필수)

**유효 기간:** 1년 (Phase 1)

**갱신:** Phase 2 시작 시 재동의

---

**Stage 2: 개별 동의 (활동 시)**

**세션별 체크리스트 (앱):**

```
섹스 세션 시작 전:

□ 파트너 동의 확인
  - 파트너: [이름 선택]
  - 데이터 공유 동의: Y/N
  
□ 드론 촬영 (선택)
  - 촬영 여부: Y/N
  - 파트너 동의: Y/N
  
□ 생체 데이터 (선택)
  - 웨어러블 착용: Y/N
  
→ 전부 확인 후 진행
```

**소요 시간:** 30초

### 케이스별 동의 프로토콜

**케이스 A: 연구원 간 섹스 (90% 예상)**

**프로세스:**
1. 앱에서 파트너 선택
2. 파트너에게 동의 요청 푸시
3. 파트너 승인 (1분 내)
4. 드론 여부 확인
5. 세션 시작

**데이터:**
- 양쪽 모두 데이터 수집
- 상호 동의 확인
- 중단 시 양쪽 모두 중단

---

**케이스 B: 외부인과 섹스 (10% 예상)**

**간소화 동의:**
1. 외부인에게 QR 코드 제공
2. QR 스캔 → 간단 동의 화면
3. "데이터 익명 공유 동의?" Y/N
4. 10초 완료

**데이터:**
- 외부인 신원 기록 안 함
- 연구원 데이터만 수집
- 외부인 데이터 선택 (대부분 거부 예상)

**중요:** 외부인 데이터는 선택이며, 연구에 필수 아님

---

**케이스 C: 드론 촬영**

**24시간 전:**
- 앱에서 촬영 신청
- 파트너 동의 확인

**당일 아침:**
- 재확인 푸시 알림
- 취소 가능

**세션 시작 전:**
- 최종 확인
- 언제든 중단 가능

**촬영 후:**
- 24시간 내 영상 확인
- 삭제 요청 가능
- 승인 시 저장

## 분석 프레임워크

### 7개 프레임워크 개요

**1. MSK (Mask Types) - 가면 유형**
- 12종 가면
- 가면 전환 패턴
- 가면 간 갈등
- 진짜 자아 vs 가면

**2. HAB (Habitus) - 아비투스**
- 4유형: HH, HL, LH, LL
- 계급 × 문화자본
- 관계에서의 역학
- 계급 이동

**3. V/H (Villain-Hero) - 악당-영웅 궤적**
- 관계 시작 → 종료 경로
- 역할 전환
- 구원 vs 파괴
- 예측 모델

**4. SEX (Sexual Spectrum) - 성적 스펙트럼**
- 주도/수용 축
- 빠름/느림 축
- 시각/촉각/청각 축
- 3D 매핑

**5. M/S (Money-Sex Correlation) - 돈-섹스 상관**
- 거래적 역학
- 경제적 불균형
- 권력 관계
- 순수 vs 거래

**6. DIV (Divorce Types) - 이혼 유형**
- 4가지 경로
- 예측 요인
- 회복 vs 재결합
- 새 관계 패턴

**7. CHG (Change Readiness) - 변화 준비도**
- 5단계 모델
- 저항 요인
- 촉진 요인
- 개입 시점
