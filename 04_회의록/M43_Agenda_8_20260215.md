### 제안: 3축 측정 시스템

#### 1-A. 척도 설계

**연속형 -1.0 ~ +1.0**

```
Dominance:
  +1.0 = 완전 지배 (명령, 통제)
   0.0 = 중립 (호혜적)
  -1.0 = 완전 복종 (순응, 따름)

Expressiveness:
  +1.0 = 극대 표현 (소리, 몸짓, 언어 활발)
   0.0 = 중립
  -1.0 = 극소 표현 (조용, 절제)

Aggression:
  +1.0 = 격렬함 (강도, 속도, 힘)
   0.0 = 온화함
  -1.0 = 수동성 (느림, 부드러움)
```

**Rafael:** "7점이나 10점 척도도 가능하지만, 연속형이 정밀하고 통계 분석에 유리합니다."

## 부록 A: 3축 상세 정의

### Leading (주도성)

| 점수 | 행동 특성 |
|:---:|----------|
| **+1.0** | 완전 주도: 명령, 통제, 지시, 페이스 결정 |
| **+0.5** | 중간 주도: 제안, 리드, 능동적 선택 |
| **0.0** | 중립: 호혜적, 교대, 협력적 |
| **-0.5** | 중간 추종: 상대 리드 따름, 수용적 |
| **-1.0** | 완전 추종: 순응, 복종, 지시 대기 |

### Expressiveness (표현성)

| 점수 | 행동 특성 |
|:---:|----------|
| **+1.0** | 극대 표현: 큰 소리, 과장된 몸짓, 언어 활발 |
| **+0.5** | 중간 표현: 적극적 반응, 소리/언어 사용 |
| **0.0** | 중립: 보통 수준 |
| **-0.5** | 중간 절제: 조용, 최소 언어 |
| **-1.0** | 극소 표현: 거의 무음, 절제, 정적 |

### Intensity (강렬함)

| 점수 | 행동 특성 |
|:---:|----------|
| **+1.0** | 극대 강렬: 격렬, 빠름, 강함, 힘 |
| **+0.5** | 중간 강렬: 활발, 에너지 있음 |
| **0.0** | 중립: 보통 속도/강도 |
| **-0.5** | 중간 온화: 느림, 부드러움 |
| **-1.0** | 극소 강렬: 매우 느림, 수동적, 최소 움직임 |

## 부록 D: 자기인식 왜곡도 계산

### 정의

```
자기인식 왜곡도 = |자가 평가 - 파트너 평가|
```

### 예시

```
Case 1:
  본인 Leading: +0.60
  파트너가 평가한 본인 Leading: +0.40
  왜곡도: |0.60 - 0.40| = 0.20 (낮음)

Case 2:
  본인 Leading: +0.30
  파트너가 평가한 본인 Leading: +0.80
  왜곡도: |0.30 - 0.80| = 0.50 (높음)
```

### 해석

| 왜곡도 | 해석 |
|:---:|------|
| 0.0-0.2 | 자기인식 정확 |
| 0.2-0.4 | 중간 왜곡 |
| 0.4-0.6 | 높은 왜곡 |
| 0.6+ | 극도 왜곡 (자기 과대/과소 평가) |

### 연구 활용

- 가면 유형과 왜곡도 상관관계
- 방어적 가면 → 과소평가 경향?
- 과시적 가면 → 과대평가 경향?
