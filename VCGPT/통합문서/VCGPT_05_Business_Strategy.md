## 1. 시장 포지셔닝 원칙

### Veilrum은 데이팅 앱으로 런칭하지 않는다

데이팅 앱으로 런칭하면 즉시 경쟁해야 하는 상대:
- Tinder / Bumble / Hinge / OkCupid

**이것은 질 수밖에 없는 시작점이다.**

대신 Veilrum은 **Relationship Intelligence Platform**으로 런칭한다.

초기 포지셔닝:
> "Understand your relationship patterns, emotions, and desires."

- Primary category: Self-understanding + relationship reflection
- Secondary category: Relationship discussion community
- Only later: Alignment discovery (매칭)

### 포지셔닝 선언

- **"관계의 Banksy"** — 자극적이지만 근거 있고, 불편하지만 반박 불가
- **카테고리 재정의:** 데이팅 앱($10B TAM)이 아닌 자기계발($40B) + 멘탈웰니스($130B) 교차점
- **핵심 메시지:** "수위를 검열하지 않는다. 합의를 검증한다."

## 2. 초기 경쟁 시장 정의

Veilrum의 초기 경쟁 시장:
- Online therapy (BetterHelp)
- Mental wellness apps (Headspace, Calm)
- Journaling platforms (Reflectly)
- Self-reflection tools

**하지만 Veilrum의 차별화:**
- Relationship-specific insight (관계 전문)
- Pattern detection (반복 패턴 탐지)
- Taboo-safe conversation (금기 안전 대화)
- Sexual relationship discussion (성적 관계 논의)

### 경쟁자 비교

| 경쟁자 | 약점 | Veilrum |
|--------|------|---------|
| BetterHelp | $260~400/월, 일반론적 | AI+데이터 맞춤형, $9.99/월 |
| Headspace | 관계에 특화 안 됨 | 관계 전문, 7개 프레임워크 |
| MBTI 앱 | 피상적 | 7개 프레임워크 × 231개 도메인 깊이 |
| Tinder | 겉모습 중심, 교육 없음 | 학습→성장→자연스러운 만남 |

### TAM 확장 논리

- 기존: "연인을 찾는 사람" → 데이팅 시장 $10B
- 재정의: "자기 자신을 이해하고 싶은 모든 사람" → 자기계발($40B) + 멘탈웰니스($130B) + 데이팅 통합

## 3. 첫 유저 문제 — 진입 트리거

**Veilrum을 처음 여는 이유는 하나다:**

> 사람은 보통 행복해서 관계 서비스를 찾지 않는다. 고통스럽거나, 혼란스럽거나, 너무 궁금할 때 들어온다.

Veilrum은 **고감정(high-emotion) 문제 진입형 제품**이다.

가장 강한 감정적 활성화 진입 시나리오 (GTM 100K 기준):
- 왜 나는 항상 같은 사람에게 끌릴까
- 이 사람은 왜 이렇게 멀어질까
- 상황썸 / 애매한 관계가 너무 힘들다
- 성적으로 너무 안 맞는 것 같다
- 관계에서 자꾸 질투/집착/불안이 올라온다
- 헤어졌는데 왜 또 같은 패턴이 반복될까
- 내가 이상한 건지 알고 싶다
- 말 못하는 성적 고민이 있다

### 3단계 전환 구조 (GTM)

#### 1단계 — 클릭 (후킹 문장)
- "왜 당신은 항상 같은 관계를 반복할까"
- "왜 불안형은 회피형에게 끌릴까"
- "왜 좋은 사람은 자꾸 지루하게 느껴질까"
- "왜 성적으로 안 맞는 관계는 말을 못 하게 될까"

#### 2단계 — 체험 (짧은 입력 후 즉각 AI 해석)
- 감정 수용 → 패턴 후보 → 짧은 인사이트 문장

#### 3단계 — 저장/복귀
- 주간 리포트 / CODETALK 루프 / "내 패턴 보기" / 내 관계 언어 지도

### 4.3 익명 스토리 플랫폼

사용자가 익명으로 관계 스토리를 포스팅할 수 있다.

- **밀라르쉬:** 얼굴 비공개 (Banksy 전략) — 목소리+실루엣/아바타/텍스트 오버레이

3. 3번째 시리즈 — 만족 단계 (미정)

## 5. 초기 제품 루프

사용자가 감정적 혼란을 안고 들어온다.

```
Write story
→ AI responds
→ user receives pattern insight
→ curiosity increases
→ deeper questions asked
→ relationship profile builds
→ weekly insight report
```

### 3가지 성장 루프

#### Insight Loop
감정/상황 입력 → AI 해석 → "정확하다" 경험 → 더 깊은 질문 → 더 많은 기록 → 더 정확한 리포트

#### Reflection Loop
하루 키워드 입력 → 100일 누적 → 개인 관계 언어 지도 생성 → 자기이해 심화 → 반복 사용

#### Community Loop
익명 경험 공유 → "나만 그런 게 아니다" 경험 → 글쓰기/댓글 → 커뮤니티 체류 증가

이 루프가 리텐션을 만든다.

### 승리 시나리오 8단계

1. 가벼운 자기이해 호기심으로 유입
2. 짧은 진단이나 기록으로 시작
3. AI가 "생각보다 정확하다"는 느낌을 줌
4. 사용자가 며칠간 기록을 이어감
5. AI가 반복 패턴을 보여줌
6. 사용자가 "내가 왜 이런지"를 이해하기 시작
7. 장기 사용으로 전환
8. 이후 민감 주제/커뮤니티/매칭으로 확장

## 6. 첫 10만 유저 성장 메커니즘

**바이럴 메커닉은 인식(Recognition)이다.**

사람들은 이럴 때 공유한다:
> "This explained my relationship perfectly."

공유 가능한 산출물:
- 관계 패턴 요약
- 페르소나 인사이트
- 갈등 패턴 설명

이것들이 소셜미디어에서 확산될 수 있다.

### 폭발 성장 포인트 5가지

1. **"나를 제일 잘 아는 AI"** — "얘는 내 패턴을 안다. 얘는 내가 반복하는 실수를 기억한다."
2. **자기이해를 게임처럼** — "나를 해석하는 게 재밌다"는 느낌
3. **관계 데이터 네트워크** — 개인 + 집단 패턴 데이터 → 관계 인텔리전스 데이터베이스
4. **"비밀스러운 나"는 엄청난 차별점** — 다른 데서는 절대 못 하는 얘기를 여기서
5. **커뮤니티 후반부 폭발** — 유저가 자기 서사를 쌓은 뒤 커뮤니티가 붙어야 강함

## 13. 성공 지표

### 제품 성공 지표 (Veilrum 정의)

단순 매칭 수가 아니라:
- 서로의 관계 기대를 얼마나 일찍 명확히 했는가
- 대화 만족도
- 일정 기간 후 관계 질 개선 여부
- 사용자가 더 자신다운 관계를 선택했는가
- 관계가 끝났더라도 더 명료하고 덜 파괴적으로 끝났는가

### 최종 성공 기준 (장기 비전)

Veilrum이 성공하면 사용자들이 보고하는 것:
- 더 깊은 자기이해
- 더 나은 관계 선택
- 향상된 소통
- 더 만족스러운 관계
- 더 강한 커뮤니티

**궁극적 지표: 인간 관계의 질 개선**

### 장기 비전

- 관계 인텔리전스의 1차 디지털 플랫폼
- 관계적 리터러시의 문화적 기준점
- 정렬된 커뮤니티의 글로벌 네트워크
- 인간 관계 패턴 이해를 위한 지식 기반

> Veilrum은 단순한 제품이 아니라, **AI 시대를 위한 진화하는 관계 생태계**다.

## 14. Sex GTM 포지션

Sex 영역은 노출 방식이 가장 중요하다.

### 초기 전면 메시지로는 쓰지 않는다
브랜드 상단은:
- 관계 이해 / 자기이해 / 관계 패턴 / 친밀감과 합의

초기에는 절대 이렇게 보이면 안 된다:
- 야한 대화 서비스 / 성인 콘텐츠 플랫폼

### 심층 유입 포인트로 활용
심층 콘텐츠/리포트에서:
- 성적 불일치 / 욕망 수치심 / 경계 / 거절과 동의 / 숨겨진 판타지
를 구조적으로 다룬다.

**Sex는 낚시용 전면이 아니라 Veilrum의 차별성을 만드는 깊이 레이어다.**

## 15. 성공 가능성 냉정 분석

### 성공 확률 추정

| 구분 | 확률 |
|------|------|
| 일반 스타트업 성공률 | 5% |
| Veilrum 구조 기준 예상 | 10~20% |

성공할 경우 파급력은 매우 크다.

### 성공 가능성이 있는 이유 3가지

1. **인간의 근본 문제를 다룬다** — 정보/생산성/엔터테인먼트가 아니라 정체성 + 관계 + 삶의 방향. 인간이 평생 고민하는 영역
2. **AI 시대에 가장 강력한 분야** — AI가 가장 강한 개인화·패턴 분석·대화 영역과 정확히 맞닿아 있음
3. **네트워크 효과 가능성** — 관계/성향/심리 데이터가 축적되면 강력한 진입장벽이 됨

### 현재 구조에서 가장 위험한 설계 3가지

**위험 1 — 너무 많은 것을 동시에**
자기 분석 + 성 분석 + 관계 분석 + 커뮤니티 + 매칭 + AI 코칭 + 인생 설계를 동시에 하려는 것.
→ **해결:** 핵심 기능 하나로 시작 — "자기 분석 + AI 코치"

**위험 2 — 성 관련 영역의 사회적 리스크**
철학적으로는 맞지만 플랫폼 운영 측면에서 정책 리스크 / 결제 리스크 / 플랫폼 규제 발생 가능.
→ **해결:** 성 영역은 초기 핵심이 아니라 확장 영역으로 배치

**위험 3 — 데이터 신뢰 문제**
자기 보고형 데이터의 한계 — 과대/과소평가, 거짓 응답.
→ **해결:** 행동 기반 분석 + 장기 데이터 + AI 패턴 분석 병행

### 폭발적 성장 핵심 포인트

1. **AI가 "나를 가장 잘 이해하는 존재"가 되는 것** — 이해받고 싶어하고, 판단받지 않기를 원하는 인간의 욕구
2. **자기 이해 중독성** — MBTI/에니어그램/성격 테스트처럼 자기이해는 강한 심리적 보상을 제공함
3. **관계 데이터 네트워크 효과** — 충분히 성장하면 세계에서 가장 큰 관계 데이터셋이 될 가능성
