## Part 2: "자극적 콘텐츠" 수위 전략

### 자극의 3축 정의

**축 1: 성적 솔직함** — 성적 패턴과 각인의 연결을 직접 언급.

**축 2: 불편한 진실** — 통계와 데이터로 관계의 환상을 깨는다.

**축 3: 금기 파괴** — 폴리아모리, 오픈 릴레이션쉬, 성적 환상, 포르노와 관계의 상관관계. 주류 미디어가 피하는 주제를 정면으로.

---

### Level 1~5 수위 스케일

**축 1: 성적 솔직함**

| Level | 내용 |
| --- | --- |
| :---: | ------ |
| 1 | 일반론 |
| 2 | 구체적 연결 |
| 3 | 유형별 패턴 직접 언급 |
| 4 | 환상-각인 연결, 구체적 행위 |
| 5 | 실제 사례의 디테일한 성적 내러티브 + 데이터 (자체 플랫폼 전용) |

**축 2: 불편한 진실**

| Level | 내용 |
| --- | --- |
| :---: | ------ |
| 1 | 상식 |
| 2 | 숫자 충격 |
| 3 | 사랑 자체에 대한 도전 |
| 4 | 개인화된 충격 |
| 5 | 데이터 기반 완전 분석 (자체 플랫폼 전용) |

**축 3: 금기 파괴**

| Level | 내용 |
| --- | --- |
| :---: | ------ |
| 1 | 중립적 인정 |
| 2 | 데이터 제시 |
| 3 | 직접 분석 |
| 4 | 판단 유보, 데이터만 |
| 5 | 실제 사례 + 데이터 + 밀라르쉬 해석 전체 (자체 플랫폼 전용) |

### 채널별 수위 매핑 (확정)

| 채널 | 역할 | 축1 성적 | 축2 진실 | 축3 금기 | 목표 |
| --- | --- | --- | --- | --- | --- |
| :---: | :---: | :---: | :---: | :---: | ------ |

### 콘텐츠 펄널 구조

```
[틱톡/릴스] 60초 훈. 불편한 진실 중심
    ↓ "더 알고 싶다면"
[유튜브] 10~20분. 사례 분석 + 프레임워크. 성적 솔직함 추가
    ↓ "날것을 원한다면"
[팟캐스트] 30~60분. 실제 대화, 편집 최소. 금기 영역 진입
    ↓ "전체 버전, 무삭제"
[자체 플랫폼] 제한 없음. 유료. 완전한 프로파일링 과정 공개
```
