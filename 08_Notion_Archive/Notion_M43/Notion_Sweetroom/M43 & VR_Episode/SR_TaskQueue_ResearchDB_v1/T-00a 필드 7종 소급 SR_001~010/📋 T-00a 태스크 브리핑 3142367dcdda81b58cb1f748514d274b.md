## 2. 컨텍스트 요약

**소급 대상 필드 7종:**

| 필드명 | 타입 | 값 예시 |
| --- | --- | --- |
| 시간대 | Select | 저녁_파티초반/중반/후반 등 |
| 장소 | Select | 스위트룸_거실_소파중앙, 바, 바닥 등 |
| 발화동기/맥락 | Text | 발화 의도 1~2문장 |
| 긴장도 | Number | 1~5 |
| 서사기능 | Select | 복선/전환/폭로/해소/강화/유머/소급/혼합 |
| 관계온도변화 | Select | ↑상승/→유지/↓하강/⚡반전 |
| 씨연결 | Relation | DB2 연결 (전체 미입력 — DB2 레코드 없음) |

**패턴 분석 (SR_001~010 전체):**

- SR_001~003: 파티 초반. 긴장도 2~3. 복선/전환 중심
- SR_004~006: 폭로 등장. SR_006 폭로 피크
- SR_007~009: 질문라운드. 폭로+해소 교차. 긴장도 5 집중
- SR_010: 야간대화. 전환+폭로 혼합. '문화적 각인 코드' + 사빈 마무리 발화로 클로징
- 전체 온도변화: ↑상승 지배적
