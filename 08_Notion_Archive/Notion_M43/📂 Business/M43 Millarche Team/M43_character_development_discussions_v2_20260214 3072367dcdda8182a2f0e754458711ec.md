# 4. 경험 출처 3가지 구분

1. **직접경험 + 공부**: "내가 겨어봤는데, 심리학적으로 이건..."
2. **간접경험 + 공부**: "내 절친이 이걸 겨었는데..."
3. **직접 + 간접 + 공부**: "나도 겨었고, 가까운 사람도 겧은 걸 봤고, 그래서 연구까지 했다"

각 캐릭터당 본인 + 파트너 + 절친 + 부모 = 최소 4~5명의 관계 데이터

11명이면 44~55명의 관계 사례가 매트릭스에 투입

# 5. 타겟 고객층 전략

핵심 타겟: 20~40대

- **30대** (1순위): 관계 문제 가장 절실, 돈도 있고 절박함도 있음
- **20대 후반** (2순위): 연애 실패 누적, 디지털 결제 거부감 없음
- **40대** (3순위): 가장 높은 객단가 가능성

문화권별 차이: 서구권(구독 모델), 동아시아(일회성 고액), 동남아/남미/중동(무료→저가 전환)
