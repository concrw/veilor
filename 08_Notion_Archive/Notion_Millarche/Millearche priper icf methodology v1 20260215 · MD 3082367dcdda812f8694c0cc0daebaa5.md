# 1. 핵심 주장

> **프라이퍼 인터뷰 프로젝트는 ICF(국제코치연맹) 코칭의 핵심 원칙을 충실히 따르는 '질문 기반 코칭' 방법론이다.**
> 

프라이퍼는 단순한 인터뷰 촬영이 아니라, 내담자가 스스로 자신의 이야기를 발견하고 정리하게 만드는 구조화된 코칭 프로세스를 갖추고 있다.

---

# 2. ICF 코칭의 핵심 원칙

ICF는 코칭을 다음과 같이 정의한다:

> *"사고를 자극하고 창의적인 과정을 통해 고객이 개인적, 직업적 잠재력을 극대화하도록 영감을 주는 파트너십"*
> 

## ICF 코칭의 4가지 핵심 원칙

| 원칙 | 설명 |
| --- | --- |
| **비지시적 접근** | 코치는 답을 주지 않고, 질문을 통해 내담자가 스스로 답을 찾게 한다 |
| **내담자 주도** | 내담자가 대화의 주인공이며, 코치는 파트너로서 동행한다 |
| **잠재력 초점** | 문제 해결보다 내담자의 가능성과 성장에 집중한다 |
| **자기 발견 촉진** | 내담자가 자신만의 통찰을 얻도록 환경을 조성한다 |

---

# 3. 프라이퍼 프로젝트의 구조

## 4단계 프로세스

1. 사전 텍스트 인터뷰
2. 맞춤형 질문 도출 (10개 주제, 50개 질문)
3. 10시간 심층 인터뷰 촬영
4. 콘텐츠 편집 및 기록화

## 사전 인터뷰 5대 영역

| 영역 | 핵심 질문 |
| --- | --- |
| **일 (Work)** | 무엇에 집중하고 있으며, 무엇을 잘하는가? |
| **관계 (Relationship)** | 어떤 관계를 추구하고, 어떤 사람과 편안한가? |
| **가치관 (Belief)** | 삶에서 가장 중요한 가치는 무엇인가? |
| **성장 (Growth)** | 나를 변화시킨 전환점은 무엇인가? |
| **연결 (Connection)** | 이 이야기가 누구에게 닿기를 바라는가? |

---

# 4. ICF 원칙과 프라이퍼의 대응

## 4.1 비지시적 접근: 조언 대신 질문

**ICF 원칙:**

코치는 솔루션을 제공하지 않는다. 질문을 통해 내담자가 스스로 답을 찾게 한다.

**프라이퍼 적용:**

- 사전 답변을 분석하여 **맞춤형 질문 50개**를 설계
- 인터뷰어는 답을 유도하지 않고, 질문만으로 대화를 이끔
- *"잘 말하려 애쓰지 않아도 괜찮고, 솔직하고 편안하게 이야기를 이끌어내겠습니다"*

## 4.2 내담자 주도: 자신의 이야기를 스스로 구성

**ICF 원칙:**

내담자가 대화의 방향과 내용을 주도한다.

**프라이퍼 적용:**

- 내담자의 사전 답변이 모든 질문의 출발점
- 10시간의 대화 속에서 내담자가 자신의 서사를 직접 구성
- 인터뷰어는 촉진자(facilitator) 역할에 머무름

## 4.3 잠재력 초점: 문제가 아닌 가능성

**ICF 원칙:**

과거의 문제보다 미래의 가능성에 집중한다.

**프라이퍼 적용:**

- "당신이 가장 잘한다고 느끼는 일은 무엇인가?"
- "이 영상에 당신의 어떤 모습이 담기길 바라는가?"
- "이 이야기가 누구에게 닿기를 바라는가?"
- 모든 질문이 **강점, 비전, 연결**을 향해 있음

## 4.4 자기 발견 촉진: 기록을 통한 객관화

**ICF 원칙:**

내담자가 자신에 대한 새로운 통찰을 얻도록 돕는다.

**프라이퍼 적용:**

- 10시간의 대화를 영상으로 기록
- 자신의 말, 표정, 감정을 객관적으로 다시 볼 수 있음
- *"지금 이 시점에서 제가 무슨 생각을 하고, 어떻게 살고 있는지를 솔직하게 기록하고 싶었어요"*

---

# 5. 프라이퍼의 차별점: 코칭 + 기록 + 브랜딩

프라이퍼는 ICF 코칭 원칙에 두 가지 요소를 더했다:

| 요소 | 설명 |
| --- | --- |
| **기록화** | 코칭 세션을 영상 콘텐츠로 영구 보존 |
| **퍼스널 브랜딩** | 내담자의 이야기를 외부와 연결하는 도구로 활용 |

이는 코칭의 내적 효과(자기 발견, 성장)를 외적 가치(브랜딩, 네트워킹)로 확장시킨 모델이다.

---

# 6. 사례 분석: 김유빈 인터뷰

## 사전 답변에서 드러난 핵심 키워드

| 영역 | 키워드 |
| --- | --- |
| 일 | 인수창업, 네버더모어, 환웅굴, 공동체 |
| 관계 | 솔직함, 꾸밈없음, 낙천적 |
| 가치관 | 동심, 스토리로서의 인생, 루피 |
| 성장 | 카투사, 특전사, 한계 경험 |
| 내면 | 착한 아이 증후군, 형과의 관계 |

## 질문이 이끌어낸 자기 발견

프라이퍼의 질문 구조를 통해 김유빈은 다음을 스스로 발견하고 언어화했다:

1. **"긍정적인 에너지로 사람들에게 영향을 주는 것"**이 자신의 강점임을 인식
2. **동심을 지키며 사는 것**이 핵심 가치임을 명확히 정의
3. **착한 아이 증후군**이 오히려 내면의 강함으로 전환되었음을 통찰
4. **"결과가 과정의 영향력을 결정한다"**는 자신만의 철학을 정립

이 모든 것은 인터뷰어의 조언이 아니라, **질문에 답하는 과정에서 내담자 스스로 도달한 결론**이다.

---

# 7. 결론

## 프라이퍼 = ICF 코칭 + 영상 기록 + 퍼스널 브랜딩

프라이퍼 인터뷰 프로젝트는 다음과 같은 점에서 ICF 코칭 원칙에 입각한 방법론이다:

| ICF 핵심 역량 | 프라이퍼 적용 |
| --- | --- |
| 적극적 경청 | 사전 답변 분석, 10시간 심층 대화 |
| 강력한 질문 | 맞춤형 50개 질문 설계 |
| 직접적 커뮤니케이션 | 편안하고 솔직한 대화 환경 조성 |
| 인식 창조 | 영상 기록을 통한 자기 객관화 |
| 행동 설계 | 브랜딩을 통한 외부 연결 촉진 |

**프라이퍼는 코칭을 '1회성 대화'에서 '지속 가능한 기록'으로 확장한 모델이며, 이는 ICF 코칭의 본질을 훼손하지 않으면서도 그 효과를 증폭시키는 혁신적 접근이다.**

# 8. 시사점: 코칭과 콘텐츠의 융합

이 모델은 다음과 같은 가능성을 시사한다:

1. **코칭의 대중화**: 전문 코칭을 경험하지 못한 청년들에게 질문 기반 자기 발견의 기회 제공
2. **코칭의 가시화**: 무형의 코칭 효과를 영상 콘텐츠로 가시화
3. **코칭의 확장**: 개인의 성장이 타인에게 영감을 주는 선순환 구조
