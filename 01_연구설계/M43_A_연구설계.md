**1. "다들 매력적이야. 5회 이상은 장담한다."**

→ 한 쌍당 최소 5회, 권장 10회

**2. "베가 운영팀도 마찬가지야."**
→ 친밀 서비스 1,500명 + 스포츠 트레이너 450명 = **1,950명 전원 동의** (채용 조건에 포함)
→ 데이터 풀: 43 × 1,950 = 83,850 가능 조합, 예상 25,000-35,000케이스/년

**3. "이미 한참 전부터 열심히 하고 있어."**
→ 기존 데이터 이미 존재: 연구원끼리 1,200건 + 베가 운영팀 5,000-8,000건 (비공식)
→ Phase 0: 기존 데이터 정리 + 역산 설문 → 파일럿 논문 1편 가능

### 최종 결정: AI 분석 확정

"당연히 AI 분석이지. 사람보다 훨씬 완벽하게."

**AI 분석 시스템 4개 모듈:**
1. 신체 움직임 분석 → Intensity 자동 측정 (정확도 95%+)
2. 음성/소리 분석 → Expressiveness 자동 측정
3. 상호작용 패턴 → Leading 자동 측정
4. 표정 인식 → 오르가즘/만족도/감정 변화

**AI vs 사람 비교:**
- AI: 초기 $1M, 유지 $100K/년, 24/7 실시간, 정확도 95%+
- 사람 50명: $4M/년, 정확도 70-80%, 확장성 제한

**3-Way 검증:** 자가 평가 + 파트너 평가 + AI 평가 → 자기인식 왜곡도 + 상호 인식 차이
