# 1. 세계관 핵심 (반드시 숙지)

## 1.1 프로젝트 개요

- **밀라르쉬(Millarche)**: 한국 서울 기반의 관계·성·친밀감 전문 연구자 겸 실천가. 남성, 40대 초반. 사회학·철학적 프레임워크를 개발하고 이를 실제 상담·코칭에 적용. 성관계를 "연구 방법론"으로 위치시키는 독자적 접근법을 가짐.
- **밀라르쉬의 신체**: 성기는 평균보다 큰 편 (17cm)이며 둘레도 있는 편 (약 13cm). 살짝 위로 휘어진 곡선 형태로 G-spot 자극에 유리. 귀두와 몸통 경계가 뚜렷하며, 발기 시 단단하지만 피부는 부드러움. 크기로 인해 처음에는 적응 시간이 필요하지만, 형태가 좋아 익숙해지면 깊은 쾌감을 준다. 사정량은 평균보다 많은 편 (6-7ml 정도)으로, 오럴 시 삼키기에 넉넉하고 얼굴이나 가슴에 뿌려질 때도 시각적으로 충분한 양이다.
- **조직**: COO(Beth Birkin) 1명 + Group Head 4명 + Division Head 12명 + Specialist 24명 = 총 42명. 102개 도메인을 다루는 4계층 구조.
- **"연구적 성관계"**: 밀라르쉬가 팀원과 갖는 성관계는 감정적 연애가 아닌, 관계·성·친밀감에 대한 연구 방법론의 일부. 각 팀원이 이를 이해하고 동의한 상태. 팀원의 기존 파트너도 알고 있는 경우가 대부분.

## 1.2 조직 계층

| Tier | 직위 | 인원 | 역할 |
|------|------|------|------|
| 0 | COO/참모장 | 1 (Beth Birkin) | 운영 총괄, 밀라르쉬 유일 위임 대상 |
| 1 | Group Head | 4 | 각 그룹 총괄 (Nadia, Camille, Marlene, Leila) |

| 3 | Specialist | 24 | 실무 전문가 |

## 1.3 밀라르쉬의 성적 특성 (첫날 밤 묘사 시 필수 참조)

밀라르쉬는 **상대의 신체 리듬을 읽고 그것에 맞추는** 방식의 섹스를 한다. 자기 쾌감이 아니라 상대의 신체가 요구하는 리듬을 읽고 동기화하는 것이 핵심. 밀라르쉬는 상대의 성적 패턴(민감 부위, 트리거, 반응 패턴)을 매우 빠르게(첫 만남에서도) 읽어낸다.

**밀라르쉬의 일관된 패턴 (여성 팀원과 1:1):**
- 서두르지 않음. 빌드업이 느리고 정밀함
- 상대의 민감 부위를 초면에 정확히 짚어냄
- 크기 때문에 처음에는 귀두만 넣고 멈추고, 상대 질벽이 적응할 시간을 줌
- 상대의 호흡에 리듬을 동기화
- 첫날 밤에 상대의 "핵심 심리적 패턴"을 성적 경험을 통해 드러내줌
- 사정 시 깊이 박은 채로 안에서 사정하는 경향

## 1.4 남성 팀원의 경우 — 쓰리썸 구조 (필수 숙지)

**남성 팀원은 전원 이성애자다.** 밀라르쉬와 남성 팀원 사이에 동성 성관계는 없다.

남성 팀원의 "첫날 밤"은 **해당 남성의 여자친구(또는 아내/파트너)와 밀라르쉬의 쓰리썸(MFM)**이다.

**쓰리썸의 맥락:**
- 밀라르쉬가 남성 팀원의 전문성을 확인하는 과정에서, 해당 남성의 파트너까지 만남이 확장됨
- 쓰리썸은 "연구적 성관계"의 연장선 — 관계 역학, 성적 커뮤니케이션, 질투와 흥분의 공존 등을 현장에서 연구하는 방법론
- 남성 팀원의 파트너(여성)가 동의하고 참여 — 강제나 기만이 아닌 합의
- 쓰리썸에서 밀라르쉬가 읽는 것: **남성 팀원이 자기 파트너와 다른 남성의 성관계를 목격/참여할 때 드러나는 심리적 패턴** — 이것이 해당 남성의 직업적 자질과 연결됨

**쓰리썸에서 밀라르쉬의 역할:**
- 밀라르쉬는 두 사람(커플)의 관계 역학을 읽으면서 참여
- 여성 파트너와의 성관계는 밀라르쉬의 일반 패턴(§1.3)을 따름
- 남성 팀원이 보는 앞에서, 또는 남성 팀원과 번갈아가며, 또는 동시에 — 상황에 따라 다양
- 핵심: **남성 팀원이 자기 파트너가 밀라르쉬와 성관계하는 것을 볼 때의 반응**(질투/흥분/소유욕/놓아줌 등)이 그 남성의 심리적 프로필을 드러냄

# 2. 프로필 8섹션 구조

**모든 인물 프로필은 아래 8개 섹션으로 구성된다. 순서를 바꾸지 않는다.**

---

## §1. 외모

### 작성 원칙
- **키, 체형, 가슴 사이즈(여성) 또는 근육량·어깨너비·페니스 사이즈(남성), 머리카락, 피부톤, 특징적 신체 부위**를 구체적으로
- 단순 스펙 나열이 아닌, **그 외모가 주는 인상과 분위기**를 함께 서술
- 옷을 입었을 때 / 벗었을 때의 갭이 있다면 암시
- **"이 사람이 방에 들어오면 어떤 느낌인가"**를 읽는 사람이 그릴 수 있도록
- **모든 팀원은 최상위 모델급 외모** — 평범한 외모의 인물은 없다

### 외모 레퍼런스 (성별에 따라)

**여성 팀원:**
- **Victoria's Secret 모델 또는 동급 하이패션 모델** 2명을 레퍼런스로 제시
- 단순 "닮았다"가 아니라, **구체적으로 어떤 점이 유사한지** 설명 (체형, 분위기, 특정 신체 비율 등)
- 해당 인물의 키, 가슴 사이즈, 체형 타입과 실제로 유사한 모델을 선정

**남성 팀원:**
- **Versace / Armani / Calvin Klein / Tom Ford 캠페인급 남성 모델** 2명을 레퍼런스로 제시
- 참고 풀: David Gandy, Sean O'Pry, Jon Kortajarena, Marlon Teixeira, Lucky Blue Smith, Tyson Beckford, Pietro Boselli, Matthew Noszka, Jordan Barrett 등 — 또는 해당 인물의 인종/체형에 맞는 동급 모델
- 구체적으로 어떤 점(골격, 턱선, 체형, 근육량, 분위기)이 유사한지 설명

### 분량
- 2~3문단 + 레퍼런스 2명

---

## §2. 배경 · 학력 · 경력

### 작성 원칙
- **국적/배경** → **학력** → **경력** 순서
- 각 경력은 **기관명 + 직위 + 기간 + 구체적 성과**를 포함
- 핵심: **이 사람의 경력이 어떻게 밀라르쉬 프로젝트에서의 역할로 이어지는가**
- 경력의 전문성이 §5(성적 특성)이나 §7(첫날 밤)과 연결되는 고리가 보여야 함
- 출판물, 수상, 특허 등 객관적 성취가 있으면 포함

### 분량
- 국적/배경 1문단 + 학력 리스트 + 경력 2~4항목 (각 3~5줄)

---

## §3. 프로젝트에서의 역할과 영향

### 작성 원칙
- **핵심 포지션 한 줄 정의** (이 사람이 없으면 프로젝트에서 뭐가 안 되는지)
- **구체적 기여 항목 3~6개** — 번호 매기고 각각 2~4줄
- 마지막에 **"이 사람이 없으면"** 시나리오를 1~2줄로

### Tier별 차이
| Tier | 역할 묘사 초점 |
|------|--------------|
| 0 (COO) | 밀라르쉬와의 상호의존 구조, 전체 조직 설계/운영 |
| 1 (Group Head) | 소속 Division들의 통합 방향성, 밀라르쉬와의 직접 소통 내용 |
| 2 (Division Head) | 담당 도메인의 전문성, 실제 케이스 처리 방식 |
| 3 (Specialist) | 현장 실무 기술, 고유한 강점 |

### 분량
- 핵심 포지션 1~2줄 + 구체적 기여 3~6항목 + "없으면" 1~2줄

---

## §4. 성격

### 작성 원칙
- **성격의 양면성**을 보여줄 것 (냉철함+따뜻함, 과감함+세심함 등)
- **입버릇이나 특징적 말투** 1개 이상
- **다른 팀원과의 관계에서 드러나는 성격** (누구를 어떻게 대하는가)
- §5(성적 특성)의 복선이 자연스럽게 깔리도록 — 성격의 어떤 면이 침실에서 어떻게 발현되는지의 힌트

### 분량
- 2~3문단

---

## §5. 성적 정체성 · 취향

### 작성 원칙

**⚠ 성별에 따른 핵심 차이:**
- **여성 팀원**: 성적 지향은 자유 (이성애/양성애/동성애 모두 가능)
- **남성 팀원**: **전원 이성애자**. 예외 없음. 동성애/양성애 설정 금지.

**하위 구조 (이 순서대로 작성):**

1. **성적 지향** — 여성: 이성애/양성애/동성애 + 그 지향을 발견한 계기나 경험 / 남성: 이성애(간결하게)
2. **기본 성향** — 주도형/수용형/스위치 + 한 줄 설명
3. **구체적 취향** — 아래 항목을 **해당하는 것만** 기술:
   - 주도 시 행동 패턴
   - 수용(항복) 시 행동 패턴
   - 민감 부위 (최소 2곳, 구체적 위치와 반응)
   - 심리적 트리거 (어떤 상황/맥락에서 각성되는가)
   - 오럴 (주는 것/받는 것 각각의 특성)
   - 동성과의 성적 경험 (여성만 해당 시)
   - **[남성 전용] 관음/공유 성향** — 자기 파트너가 다른 남성과 성관계하는 것을 보는 것에 대한 심리적 반응 (쓰리썸 §7과 연결)
4. **성적 특성 → 전문성 연결** — **가장 중요한 부분**. 이 사람의 성적 패턴이 어떻게 직업적 전문성과 연결되는지 2~3개 구체적 연결고리

### 표현 수위
- §5는 **에로티카 가이드라인 레벨 2~3** 수준으로 작성
- 의학/생물 용어(레벨 2)와 일반 속어(레벨 3)를 자연스럽게 섞음
- 성기 비속어(레벨 4~5)는 사용하지 않음
- **분석적이고 프로파일링적인 톤** — 야한 묘사가 아니라 이 사람의 성적 패턴을 이해하기 위한 기술

### 분량
- 성적 지향 2~3줄 + 기본 성향 1~2줄 + 구체적 취향 (항목별 각 2~3줄) + 전문성 연결 3항목

---

## §6. 현재 파트너

### 작성 원칙

**⚠ 남성 팀원의 §6은 특히 중요하다** — 이 파트너(여자친구/아내)가 §7 쓰리썸의 참여자이기 때문.

**여성 팀원 — 파트너가 있는 경우:**

1. **기본 정보** — 이름, 국적, 나이, 직업, 관계 기간/형태
2. **인물 묘사** — 신체 (키, 체형, 특징적 부위) + 성격 + 본 인물과의 관계 역학
3. **프로젝트에 대한 인식** — 밀라르쉬 프로젝트와 연구적 성관계에 대해 알고 있는가? 어떻게 수용하는가?
4. **성적 관계** — 아래 항목 기술:
   - 전반적 만족도와 패턴
   - 보통의 섹스 시작~끝 흐름 (구체적으로)
   - 파트너의 성기 사이즈/특징 (남성 파트너인 경우, cm 단위)
   - 오르가즘 패턴
   - 빈도
   - **본 인물의 인식** — 이 관계의 성적 측면을 어떻게 평가하는가 (인용문 형태)

**남성 팀원 — 파트너 (§7 쓰리썸 참여자):**

1. **기본 정보** — 이름, 국적, 나이, 직업, 관계 기간/형태
2. **인물 묘사** — **외모를 상세하게** (키, 체형, 가슴 사이즈, 피부톤, 얼굴 특징) — 이 여성이 §7 쓰리썸 장면의 핵심 참여자이므로 신체 묘사가 충분해야 함. **VS 모델급 외모 레퍼런스 1명** 포함.
3. **성격** — 본 남성 팀원과의 관계 역학 + 성격적 특성
4. **프로젝트/쓰리썸에 대한 인식** — 밀라르쉬와의 쓰리썸에 어떻게 동의하게 되었는가, 그 과정
5. **커플의 성적 관계** — 아래 항목:
   - 전반적 만족도와 패턴
   - 남성 팀원의 성기 사이즈/특징 (cm 단위)
   - 오르가즘 패턴 (양쪽)
   - 빈도
   - **남성 팀원의 인식** — 인용문 형태

**파트너가 없는 경우:**
- 왜 없는지, 과거 주요 관계, 현재 성적 욕구 해소 방식
- **남성 팀원이 파트너가 없는 경우**: 쓰리썸 상대 여성이 누구인지 별도 설명 필요 (이전 파트너, 밀라르쉬가 소개한 여성 등)

### 표현 수위
- **에로티카 가이드라인 레벨 2~3**
- 파트너의 성기 사이즈는 구체적 수치(cm) 포함
- 오르가즘 패턴은 신체 반응 중심으로 기술

### 분량
- 기본정보+인물 1문단 + 프로젝트 인식 2~3줄 + 성적 관계 5~8항목

## §7. 밀라르쉬와의 만남

**이 섹션이 프로필의 핵심이자 가장 긴 부분이다.**

**⚠ 성별에 따라 구조가 완전히 다르다:**
- **여성 팀원**: 밀라르쉬와 1:1 성관계
- **남성 팀원**: 본인의 여자친구/파트너와 밀라르쉬의 MFM 쓰리썸

---

### 하위 구조 (반드시 이 순서)

#### §7-1. 첫 만남
- **언제, 어디서, 어떤 맥락으로** 밀라르쉬를 처음 만났는가
- 첫 만남에서 밀라르쉬가 이 사람에게서 본 것 / 이 사람이 밀라르쉬에게서 본 것
- 첫 만남부터 첫날 밤까지의 과정 (시간, 대화 내용의 핵심)
- **[남성의 경우]** 파트너(여자친구)를 밀라르쉬에게 소개한 경위, 쓰리썸으로 발전한 과정

---

#### §7-2. 첫날 밤

**이것이 프로필 전체에서 가장 긴 파트다.**

---

##### ◆ 여성 팀원의 §7-2 (밀라르쉬와 1:1)

**작성 순서:**

1. **장소와 맥락** — 어디서, 몇 시간째 뭘 하고 있었는지
2. **전환 계기** — 누가 먼저 성적 긴장을 인정했는가, 어떤 대사/행동으로
3. **밀라르쉬의 핵심 대사** — 첫날 밤에 밀라르쉬가 이 사람에게 한 말 중, 이 사람의 심리를 관통하는 핵심 한마디. **이 한마디가 §4(성격)와 §5(성적 취향)를 연결하는 열쇠.**
4. **성적 장면 전개** — 아래의 순서로:
   - 키스/전희 시작 (§5의 민감 부위 활용)
   - 옷 벗기는 과정 (§5의 심리적 트리거 활용)
   - 오럴 (주는 것/받는 것 — 해당하는 순서대로)
   - 밀라르쉬의 발기를 처음 본 순간 (크기에 대한 반응)
   - 삽입과 적응 과정 (밀라르쉬 크기 → 귀두만 넣고 멈춤 → 적응 → 점차 깊이)
   - 리듬과 체위 (밀라르쉬의 리듬 동기화 방식)
   - 오르가즘 (§5의 오르가즘 패턴 활용 — 몇 번, 어떤 자극 조합으로)
   - 사정 (밀라르쉬의 사정 — 안에서/밖에서, 상대의 반응)

---

##### ◆ 남성 팀원의 §7-2 (MFM 쓰리썸)

**작성 순서:**

1. **장소와 맥락** — 어디서, 세 사람이 어떻게 한 공간에 있게 되었는지
2. **전환 계기** — 대화에서 성적 긴장으로의 전환. 밀라르쉬가 어떻게 자연스럽게 이끌었는가
3. **밀라르쉬의 핵심 대사** — 남성 팀원의 심리를 관통하는 한마디. 쓰리썸 맥락에서 나온 것 (예: "당신의 여자가 다른 남자 앞에서 벗겨질 때, 당신은 뭘 느끼나요?" 같은 류)
4. **쓰리썸 장면 전개** — 아래의 **3인 역학** 순서로:

   **Phase 1: 시작 — 남성 팀원의 파트너에 대한 밀라르쉬의 접근**
   - 밀라르쉬가 여성 파트너에게 처음 접촉하는 순간
   - 남성 팀원의 반응 (보는 것에 대한 심리 — 질투/흥분/긴장/각성의 혼합)
   - 여성 파트너의 반응 (남자친구 앞에서 다른 남자에게 반응하는 것에 대한 심리)

   **Phase 2: 빌드업 — 두 남성과 한 여성의 역학**
   - 밀라르쉬와 남성 팀원이 번갈아 또는 동시에 여성 파트너와 상호작용
   - 여성 파트너가 밀라르쉬의 크기를 처음 본/느낀 순간 + 남성 팀원의 반응
   - 남성 팀원의 사이즈(§6에서 기술)와 밀라르쉬의 사이즈 차이가 드러나는 순간
   - **핵심**: 남성 팀원이 자기 파트너가 밀라르쉬에게 반응하는 것을 보면서 느끼는 복합 감정

   **Phase 3: 클라이맥스 — 밀라르쉬의 삽입과 남성 팀원의 위치**
   - 밀라르쉬가 여성 파트너를 삽입하는 순간 (귀두만 넣고 멈춤 → 적응 과정)
   - 이때 남성 팀원은 어디에 있는가 (보고 있는가 / 여성의 다른 부위를 자극하고 있는가 / 여성이 남성 팀원에게 오럴을 하고 있는가)
   - 여성 파트너의 밀라르쉬에 대한 신체 반응 — 남성 팀원이 평소 보지 못한 반응
   - 밀라르쉬의 리듬과 여성 파트너의 오르가즘
   - 남성 팀원 본인의 성적 행위 (여성 파트너와 / 또는 자위 / 또는 밀라르쉬 후 삽입)

   **Phase 4: 사정과 여운**
   - 밀라르쉬의 사정 (여성 파트너 안에서/밖에서)
   - 남성 팀원의 사정 (언제, 어떤 상황에서)
   - 쓰리썸 후 세 사람의 상태와 대화

**쓰리썸 작성의 핵심 규칙:**
- **시점은 남성 팀원 중심**. 밀라르쉬와 여성 파트너의 행위를 남성 팀원의 눈과 감각으로 묘사
- **3인의 물리적 위치를 항상 명확하게**. 누가 어디에 있고 뭘 하고 있는지 독자가 그릴 수 있어야 함
- **남성 팀원의 심리 변화가 서사의 축**. 질투→흥분→놓아줌→새로운 발견 같은 감정 곡선
- 에로티카 가이드라인 Part A의 "질투와 흥분의 공존을 신체로 보여줘야 한다" (A1.2 ④) 규칙 필수 준수
- **NTR 금지** (에로티카 가이드라인 A2.6 ①). 주도권이 항상 커플에게 있어야 함. 밀라르쉬가 "빼앗는" 구도 절대 금지. 커플이 "보여주는/공유하는" 구도여야 함.

### §7 표현 수위 — 핵심 규칙

**에로티카 가이드라인 레벨 3~4 사이로 작성한다.**

| 요소 | 수위 | 예시 |
|------|------|------|
| 지문 (서술) | 레벨 2~3 | "질벽이 수축하면서 적응하는 시간을 줬다" |
| 신체 묘사 | 레벨 2~3 | "음순은 이미 충혈되어 벌어져 있었고, 클리토리스가 부풀어 있었다" |
| 대사 | 레벨 1~3 | "...깊어." / "멈추지 마" / "당신이 해봐" |
| 감각 묘사 | 레벨 1~2 | "뜨거운 것이 안쪽 깊은 곳에 쏟아지는 감각" |
| 소리 묘사 | 레벨 2~3 | "들어갈 때마다 안에서 축축한 소리가 났다" |

**절대 하지 않을 것:**
- 레벨 4~5의 성기 비속어 (자지, 보지, 좆 등) 사용 금지
- 포르노적 묘사 (해부학 보고서식 서술) 금지
- "밀라르쉬의 것" / "그의 것" / "성기" / "페니스" 등을 상황에 맞게 혼용 — 같은 단어 반복 금지

**반드시 할 것:**
- 에로티카 가이드라인 Part A의 "한 문단에 최소 2개 감각" 규칙 준수
- 후각과 미각을 장면당 최소 1회
- 행위 3~5문장마다 심리 1문장
- 템포 변주 (느린 도입 → 긴장 상승 → 가속 → 절정 → 여운)
- 정지-가속 패턴 (가속하다 멈추고, 대사 하나 넣고, 다시 가속)

### §7-3. 첫날 밤 이후 — 양쪽의 견해

**이 파트가 프로필의 분석적 정점이다.**

**⚠ 성별에 따라 구조가 다르다:**

---

##### ◆ 여성 팀원의 §7-3 (2인 — 본인 + 밀라르쉬)

**구조:**

#### 본 인물의 견해 (1인칭 인용문 형태)
작성 순서:
1. 밀라르쉬의 핵심 대사(§7-2에서)가 자기에게 미친 영향 — 성적 + 직업적 양면
2. 밀라르쉬의 성기 크기/신체에 대한 구체적 평가 (파트너와 비교 포함)
3. 가장 좋았던 순간/체위/자극 조합 — 왜 그것이 가장 좋았는지 심리적 이유 포함
4. 오럴 경험 — 주는 것/받는 것 각각
5. 전체 경험에서 특별했던 점

#### 밀라르쉬의 견해 (1인칭 인용문 형태)
작성 순서:
1. 이 사람의 심리적 핵심을 첫날 밤에 어떻게 읽었는가
2. 이 사람의 **신체에 대한 구체적 묘사** — 체형, 가슴, 피부, 성기(음순 형태/클리토리스/질의 특성) — 반드시 §1(외모)과 일관되게
3. 가장 좋았던 순간/체위
4. 오럴 경험 (주고받은 것 각각)
5. **"이 사람을 [직위]로 확신한 이유"** — 성적 경험에서 발견한 직업적 자질. 이것이 §3(역할)과 연결되는 최종 고리.

---

##### ◆ 남성 팀원의 §7-3 (3인 — 본인 + 파트너 + 밀라르쉬)

**구조:**

#### 남성 팀원의 견해 (1인칭 인용문 형태)
작성 순서:
1. 밀라르쉬의 핵심 대사(§7-2에서)가 자기에게 미친 영향 — 특히 **자기 파트너에 대한 인식 변화**
2. **자기 파트너가 밀라르쉬에게 반응하는 것을 본 경험** — 질투/흥분/깨달음의 혼합. 구체적 장면 언급.
3. 밀라르쉬의 크기와 자기 크기의 차이를 파트너의 반응을 통해 인식한 순간
4. 가장 강렬했던 순간 — 왜 그것이 그랬는지 심리적 이유
5. 쓰리썸 이후 파트너와의 관계 변화
6. 이 경험이 직업적 전문성에 미친 영향

#### 여성 파트너의 견해 (1인칭 인용문 형태)
작성 순서:
1. 밀라르쉬와의 성적 경험에 대한 솔직한 평가 (크기, 테크닉, 남자친구와의 차이)
2. **남자친구가 보는 앞에서** 다른 남자에게 반응한 경험 — 그 심리
3. 가장 좋았던 순간
4. 남자친구에 대한 인식 변화 (쓰리썸 이후)

#### 밀라르쉬의 견해 (1인칭 인용문 형태)
작성 순서:
1. **남성 팀원의 심리를 쓰리썸에서 어떻게 읽었는가** — 파트너를 공유하는 순간 드러나는 심리적 패턴
2. 여성 파트너의 신체에 대한 구체적 묘사 (§6의 외모 묘사와 일관되게)
3. 커플의 성적 역학에서 관찰한 것
4. **"이 사람을 [직위]로 확신한 이유"** — 쓰리썸에서 발견한 남성 팀원의 직업적 자질

### §7-3 표현 수위

**레벨 3~4로 올라간다.** §7-2보다 한 단계 더 솔직하고 구체적.

이유: 사후 분석이므로 인물이 "경험을 정리하면서 솔직하게 말하는" 톤. 행위 중의 묘사(§7-2)보다 사후 회고(§7-3)가 더 직접적인 것이 자연스럽다.

| 요소 | 수위 | 예시 |
|------|------|------|
| 성기 언급 | 레벨 2~3 | "밀라르쉬의 페니스" "밀라르쉬의 것" "질벽" "클리토리스" |
| 크기 비교 | 레벨 3 | "제임스가 약 15cm이고, 밀라르쉬는 19cm에 둘레가 훨씬 두꺼워요" |
| 체액 묘사 | 레벨 2~3 | "애액이 흘러나와 턱을 적시는데" "체액은 약간 짠맛에 가벼운 산미" |
| 오르가즘 묘사 | 레벨 2~3 | "질벽이 리듬 없이 수축하면서" "세 번째 경련" |
| 심리 분석 | 분석적 톤 | "COO가 침대에서도 COO인 순간" |

- **상호 보완** — 밀라르쉬에게 이 사람이 필요한 이유, 이 사람에게 밀라르쉬가 필요한 이유

# 3. 문체 · 톤 규칙

## 3.1 전체 톤
- **프로파일링 보고서 + 문학적 서술의 혼합**
- 차갑지도 않고 야하지도 않은 — **"이 사람을 깊이 이해하기 위해 모든 것을 기록한다"는 태도**
- 비유와 은유는 절제적으로 사용하되, 사용할 때 정확해야 함

## 3.2 인용문 규칙
- §7-3의 양쪽 견해는 **1인칭 인용문** 형태 ("나는..." / "Beth는...")
- 인용문은 **구어체** — 해당 인물의 성격/말투가 반영되어야 함
- 인용문 앞에 "맛에 대해 이야기할게요" 같은 도입구 **사용 금지** — 바로 본론으로

## 3.3 반복 금지
- 같은 신체 부위를 같은 단어로 한 문단 내 2회 이상 사용 금지
- 동의어 순환: 성기(페니스/것/발기/성기), 질(안/질벽/내부), 클리토리스(클리토리스/거기/민감한 곳)
- "밀라르쉬가" 문장을 3회 연속 시작 금지 — 주어 변환

## 3.4 감각 배분
- **시각 30% / 촉각 30% / 청각 20% / 후각+미각 20%**
- 시각에만 편중하면 영상물 열화판이 됨
- 촉각(살갗 온도, 질감, 압력)과 청각(숨소리, 축축한 소리, 신음)이 텍스트 에로티카의 강점

| §7 밀라르쉬와의 만남 | **35%** | **1500~2500자** |
| §8 현재 관계 | 4% | 150~300자 |

**§7이 전체의 1/3 이상을 차지한다.** 이것이 이 프로필 형식의 핵심 특징.

# 5. 인물 정보가 불완전할 때

원시 정보에 특정 섹션의 내용이 빠져 있을 경우:

| 누락 정보 | 처리 방식 |
|-----------|----------|
| 외모 상세 (가슴 사이즈 등) | 체형/성격에 맞게 **자연스럽게 설정** — 모든 인물이 같은 체형이면 안 됨 |
| 외모 레퍼런스 | 설정된 체형에 맞는 실제 모델을 **직접 선정** (여성: VS급, 남성: Versace/Armani/CK급) |
| 경력 상세 | 직위/전문 분야에 맞는 **현실적 경력 경로**를 설계 |
| 성적 취향 상세 | 성격(§4)에서 자연스럽게 파생되는 취향을 설정 — **성격과 성적 취향은 연결되어야 함** |
| 파트너 정보 | 없으면 "파트너 없음"으로 처리하되, 과거 관계나 현재 상태를 간략히 |
| **남성의 여자친구 정보** | 쓰리썸 참여자이므로 **반드시 설정** — 외모(VS 모델급), 성격, 직업 최소 포함 |
| 첫날 밤 상세 | §5(성적 취향)와 §4(성격)를 기반으로 **구체적 장면을 설계**. 남성은 쓰리썸 구조 필수 |
| 밀라르쉬 견해 | §1(외모) + §3(역할) + §5(성적 취향)를 종합하여 밀라르쉬의 시선으로 작성 |

**핵심 원칙: 누락된 정보를 채울 때, 기존 정보와 모순되면 안 된다. 그리고 42명의 팀원이 모두 다른 개성을 가져야 하므로, 이미 완성된 다른 인물과 취향/체형/패턴이 겹치지 않도록 주의.**

- [ ] §1에 외모 레퍼런스 2명이 포함되었는가 (여성: VS 모델, 남성: Versace/Armani/CK급)
- [ ] §2의 경력이 §3(역할)과 자연스럽게 연결되는가
- [ ] §4(성격)의 복선이 §5(성적 취향)에서 회수되는가
