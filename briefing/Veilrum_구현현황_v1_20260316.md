> 작성일: 2026-03-16 | 대상: 이 프로젝트를 처음 파악하려는 사람 또는 새 세션 시작 시

---

## 1. 무엇을 만드는가

**Veilrum** — 관계 자기이해 플랫폼.

"비밀스러운 나를 평생 숨긴 채로 사는 것이 진짜 행복인가?"라는 질문에서 출발해, 유저가 자신의 관계 패턴을 발견하고 관리하고 만족에 이르도록 돕는 앱이다.

핵심 구조: **하나의 Veilrum 계정 → 4개 모듈이 각각의 인터페이스로 열림**

```
Veilrum 계정 (단일 RI API)
  ├── PRIPER    — 관계 유형 진단 (발견)
  ├── CODETALK  — 일일 관계 키워드 코드토크 (발견 → 관리)
  ├── DIVE      — 심층 상담 (관리)
  └── Community — 유형별 그룹 커뮤니티 (만족의 전제)
```

슈퍼앱이 아니다. 유저 데이터는 공유되지만 경험은 각 모듈별로 완전히 분리된다.

---

## 2. 유저 여정 — 발견 → 관리 → 만족

모든 기능이 이 3축으로 정렬된다.

| 단계 | 기능 | 수익 모델 |
|------|------|---------|
| **발견** | PRIPER (가면 진단) | 무료 훅 |
| **발견 → 관리** | CODETALK (100일 키워드) | 무료 → 구독 전환점 |
| **관리** | DIVE (심층 상담) | 구독 $9.99/월 |
| **만족** | Community + 합의 결과 측정 | 프리미엄 업셀 |

인생 전환점(결혼, 출산, 이직, 이별 등)마다 발견→관리→만족 사이클이 재시작된다 → 이탈 구조적 방지.

## 3. 모듈별 목표 기능

### PRIPER — 관계 유형 진단
- MSK 가면 12종 분류 기반 **30문항 진단**
- 진단 결과: 가면 지도 시각화 (레이더 차트 또는 유사 형태)
- 결과를 바탕으로 DIVE·CODETALK에 개인화 데이터 제공

### CODETALK — 100일 관계 코드토크
- DB에 저장된 100개 키워드를 하루 1개씩 제공
- 유저가 키워드에 대한 자신의 생각을 기록
- 100일치 누적 데이터 → 관계 패턴 분석의 원료

### DIVE — 심층 상담
- 유저가 관계 질문을 자유 입력
- **매칭 알고리즘**: 1,322개 Q&A DB에서 유사 질문 탐색 → 연구원 페르소나 답변 반환
  - 매칭 공식: `키워드 스코어 × 0.5 + 카테고리 스코어 × 0.2 + Jaccard 유사도 × 0.3`
  - 임계값 0.25 이상, 상위 5개 반환
- F모드(감정 지지) / T모드(분석) 구분 예정
- 모든 질문-매칭 로그 자동 기록 (`m43_user_question_logs`)

### Community — 그룹 커뮤니티
- 24개 그룹 사전 설계 완료 (DB 저장)
- 유저 유형별 그룹 자동 배정
- 포럼 + DM 기능

## 6. 현재까지 구현된 것

### ✅ 완료

| 인증 구조 | Zustand `authStore` + `ProtectedRoute` |
| 라우팅 | `/login` → `/` → `/dive` (기본 구조) |
| **DIVE 매칭 알고리즘** | `src/services/diveService.ts` — 1,322개 Q&A 매칭 로직 완성 |
| **DivePage UI** | `src/features/dive/DivePage.tsx` — 질문 입력 + 매칭 목록 + 답변 상세 |
| LoginPage | 껍데기 (로그인 폼 미구현) |
| HomePage | 껍데기 ("홈" 텍스트만) |

### DIVE 알고리즘 상세 (완료된 핵심)

```
diveMatch(userQuestion)
  → tokenize (한국어 형태소 분리)
  → DB에서 m43_domain_questions 전체 로드 (도메인·연구원·답변 조인)
  → 각 질문에 대해 score 계산:
      keyword_score × 0.5  (DB keywords[] vs 유저 쿼리 토큰)
      category_score × 0.2 (12개 카테고리 키워드 힌트)
      text_score × 0.3     (Jaccard 유사도)
  → score ≥ 0.25인 것만, 상위 5개 반환
  → m43_user_question_logs에 자동 INSERT
```
